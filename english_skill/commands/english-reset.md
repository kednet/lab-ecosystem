# english-reset — сброс прогресса

Сбрасывает уроки конкретной недели или весь прогресс. Требует подтверждения (или `--force`).

## Использование

```bash
# Показать что будет сброшено
python scripts/english.py reset

# Сбросить конкретную неделю
python scripts/english.py reset --week=3

# Сбросить ВСЁ
python scripts/english.py reset --all

# Без подтверждения
python scripts/english.py reset --all --force
python scripts/english.py reset --week=3 --force
```

## Параметры

| Параметр | Описание |
|---|---|
| `--week=N` | Сбросить только неделю N |
| `--all` | Сбросить весь прогресс |
| `--force` | Не спрашивать подтверждение |

## Поведение

**Без аргументов:**
- Показывает текущее состояние
- Предлагает `--week=N` или `--all`

**`--week=N`:**
- Печатает список уроков недели N, которые будут удалены
- Спрашивает подтверждение
- Удаляет `state/lessons/<N>_*` файлы
- Если `current_week == N` → откатывает на `day=1`
- **Не** трогает quiz_scores
- **Не** трогает streak

**`--all`:**
- Печатает всё что будет удалено (lessons, scores, streak, progress.json)
- Спрашивает подтверждение
- Удаляет ВСЁ из state/
- После `reset --all` нужно заново `start`

## Защита от случайного сброса

- Все операции требуют подтверждения (кроме `--force`)
- Печатается список того, что удалится
- **Нет backup** — это сознательное решение, см. [[../sub-skills/progress-tracking]]

## Когда использовать

- **`--week=N`:** если чувствуешь, что не до конца понял блок (W3 = Present Perfect vs Past Simple), пройди заново
- **`--all`:** если хочешь начать с нуля после долгого перерыва
- **НЕ использовать:** для "проверить что будет" — `reset` без аргументов показывает state без удаления
