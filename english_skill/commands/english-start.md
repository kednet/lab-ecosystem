# english-start — инициализация English Skill

Создаёт `state/progress.json` и показывает план на 12 недель.

## Использование

```bash
# Первый запуск
python scripts/english.py start

# Переинициализация (удалит ВЕСЬ прогресс)
python scripts/english.py start --force
```

## Что делает

1. Создаёт `state/progress.json` с B1, IT/business goal, current_week=1
2. Создаёт пустую директорию `state/lessons/`
3. Печатает таблицу всех 12 недель (тема + грамматика)
4. Печатает "Что делать сейчас" — следующие 4 шага

## Идемпотентность

- **Без `--force`:** если уже зарегистрирована — показывает текущее состояние, не сбрасывает
- **`--force`:** удаляет `state/progress.json` и все `state/lessons/*.json`, начинает с нуля

## Когда использовать

- **Первый раз** — безусловно при старте
- **Сброс застрявшего прогресса** — `--force` если хочешь начать заново
- **После долгого перерыва** — НЕ сбрасывай, streak можно восстановить
