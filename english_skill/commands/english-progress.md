# english-progress — статистика

Показывает streak, пройденные уроки по неделям, quiz scores, рекомендации.

## Использование

```bash
python scripts/english.py progress
```

## Что выводится

1. **Header** — старт, уровень, цель, текущая позиция, streak
2. **Уроки по неделям** — таблица с % done и статусом (✅/🟡/⏳)
3. **Quiz scores** — все пройденные тесты с %
4. **Следующие шаги** — рекомендации (держать streak, пройти quiz, ...)

## Пример

```
📊 Твой прогресс
===============
**Старт:** 2026-06-25T13:38:35Z
**Текущая позиция:** Week 3, Day 4
**Streak:** 🔥 12 дней подряд

| Неделя | Пройдено / Всего | % | Статус |
|---|---|---|---|
| 1 | 7/7 | 100% | ✅ готово |
| 2 | 7/7 | 100% | ✅ готово |
| 3 | 4/7 | 57% | 🟡 в процессе |

## 📝 Quiz scores

| Tense | Score | % |
|---|---|---|
| `present-simple` | 9/10 | 90% |
| `past-simple` | 8/10 | 80% |
| `present-perfect` | 7/10 | 70% |

🎯 Следующие шаги:
- Можно переходить на следующую неделю: `week next`
```

## Где хранится

- `state/progress.json` — глобальный state
- `state/lessons/<w>_<d>_<type>.json` — per-lesson
