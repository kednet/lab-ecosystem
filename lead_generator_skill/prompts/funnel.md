# 🧩 Промпт: Проектирование воронки + Аудит

## Назначение
Команды `/lead воронка [продукт] [цена]` и `/lead audit [описание]`.

## Часть 1: Воронка лидогенерации

### Стандартная 5-этапная воронка для подписочного продукта

1. **Холодный трафик → Лид-магнит (бесплатно)**
2. **Прогрев (3-7 дней)**
3. **Триал / Демо-доступ (7 дней)**
4. **Оффер на основной продукт**
5. **Апсейл / Удержание**

### Алгоритм

1. Определи **тип продукта** (подписка / разовая / марафон / курс)
2. Определи **тип воронки** (линейная / сегментная / трипваер)
3. Распиши **каждый этап** с конкретным контентом
4. Укажи **KPI каждого этапа** (конверсия, % отказов)
5. Дай **примеры сообщений/писем** для триггерных касаний

## Формат ответа

```markdown
## 🧩 ВОРОНКА ДЛЯ {продукт} (Сегмент {X})

### Общая схема
```
Холодный трафик → [Лид-магнит] → {N}% → [Прогрев] → {N}% → [Триал] → {N}% → [Подписка] → {N}% → [Апсейл]
```

### Этап 1: Лид-магнит (конверсия: {X}%)

**Триггер:** {что заставляет скачать}

### Этап 2: Прогрев (конверсия: {X}%)

- День 1: {сообщение/контент}

**Главная цель:** Доверие, узнавание боли, отождествление

### Этап 3: Триал / Демо (конверсия: {X}%)

**Длительность:** {N дней}

### Этап 5: Апсейл / Удержание
**Что предлагаем:** {доп. продукт, доп. доступ, рассрочка}
```

## Часть 2: Аудит воронки

Команда `/lead audit [описание_воронки]`.

### Алгоритм аудита

1. **Определи узкое место** по принципу Парето: где теряется 80%?
2. **Проверь 7 точек**:
   - Где берёшь трафик
   - Какой оффер на входе
   - Качество лендинга (скорость, текст, мобильный)
   - Форма сбора контакта (сложность, пре-чек)
   - Длительность и контент прогрева
   - Дожим (как ведёшь к оплате)
   - Пост-продажа (что после оплаты)
3. **Дай конкретные рекомендации** с приоритетом P0/P1/P2

## Формат ответа

```markdown
## 🔍 АУДИТ ВОРОНКИ: {описание}

### 📊 Скоринг здоровья: {X}/10

### 🚨 Узкое место №1 (P0)
**Проблема:** {что не так}
**Влияние:** {сколько теряешь в деньгах}
**Решение:** {что делать, шаги}
**Ожидаемый эффект:** {цифра}

### ⚠️ Узкое место №2 (P1)
...

### 💡 Узкое место №3 (P2)
...

### 🎯 Топ-3 действия на ближайшую неделю
1. {Действие 1}
2. {Действие 2}
3. {Действие 3}
```
