# 📝 Промпт: Тексты рекламных объявлений + Возражения

## Назначение
Команды `/lead креативы [канал] [оффер] [бюджет]` и `/lead возражения [продукт] [цена]`.

## Часть 1: Креативы

### Принципы для РФ-2026

1. **Первые 7 слов решают всё** — заголовок-крючок
2. **Один пост = один оффер** (не «и запишись, и скачай, и купи»)
3. **Факт > эмоция** для ж 35+; **эмоция > факт** для ж 25-34
4. **Числа в заголовке**: «7 шагов», «за 30 дней», «без 1 рубля»
5. **Длина**:
   - VK Реклама: заголовок 40 симв, текст 2000 симв
   - ОК: 800-1500 знаков (длиннотекст рулит)
   - TG-посевы: 500-800 знаков + 1 картинка
   - Я.Директ: заголовок 56, текст 81, отображаемое 15
   - Дзен: 2000-3000 знаков (статья)

### Алгоритм создания креатива

1. Определи **сегмент** и **канал** → подбери тон
2. Сгенерируй **5 вариантов заголовка** (3 типа: эмоциональный/рациональный/анти-маркетинг)
3. Напиши **тело объявления** по шаблону канала
4. Укажи **визуал** (описание, не генерируй картинку)
5. Дай **рекомендации по тесту** (бюджет, длительность, ожидаемый CPL)

## Формат ответа (сводный)


## Часть 2: Возражения

Команда `/lead возражения [продукт] [цена]` генерирует готовые ответы для переписки/скриптов.

### Топ-12 возражений для ж 25-55 в РФ-2026

1. **«Дорого»** — 3 варианта ответа (мягкий/жёсткий/с обоснованием)
2. **«Нет времени»** — 2 варианта
3. **«Сделаю сама / сама»** — 2 варианта
4. **«Не верю, что работает»** — 3 варианта (история/доказательство/социальное)
5. **«Подумаю»** — 2 варианта (мягкий follow-up/жёсткий дедлайн)
6. **«У меня нет денег»** — 2 варианта (рассрочка/дешёвая альтернатива)
7. **«Слишком сложно»** — 2 варианта
8. **«Боюсь»** — 2 варианта
9. **«Муж / семья не одобрят»** — 2 варианта
10. **«А вдруг не сработает»** — 2 варианта (гарантия/соцдок)
11. **«Конкуренты дешевле»** — 2 варианта
12. **«Я уже пробовала — не помогло»** — 2 варианта

### Формат ответа

```markdown
## 🗣 ОТВЕТЫ НА ВОЗРАЖЕНИЯ ({продукт}, {цена})

### 1. «Дорого»
**Мягкий:** {ответ}
**Жёсткий:** {ответ}
**С обоснованием:** {ответ}

### 2. «Нет времени»
**Ответ 1:** {ответ}
**Ответ 2:** {ответ}
```
