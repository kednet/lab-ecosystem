# 🎯 Промпт: Анализ целевой аудитории

## Назначение
Команда `/lead анализ [сегмент]`. Создаёт глубокий портрет одного сегмента ЦА.

## Алгоритм работы

1. **Определи сегмент** по умолчанию (A: 25-34, B: 35-44, C: 45-55, D: 55-65).
2. Спроси уточнения, если данные неполные (гео, доход, ниша, продукт).
3. Сгенерируй портрет по 7 блокам.
4. Укажи 3-5 точек входа + 5-7 ключевых запросов.

## Формат ответа

```markdown
## 👤 ПОРТРЕТ СЕГМЕНТА {буква}: {название}

### 📊 Демография
- **Пол:** женский (95%)
- **Возраст:** {диапазон}
- **Гео:** {города/регионы}
- **Доход:** {уровень}
- **Семейное положение:** {статус}
- **Образование:** {если важно}
- **Занятость:** {тип}

### 🧠 Психография
- **Интересы:** {5-7}
- **Ценности:** {3-5}
- **Образ жизни:** {типичный день}
- **Кто влияет на решения:** {подруги, мама, муж, блогер}
- **Страхи:** {3-5}

### 💔 Боли и проблемы
- **Острая боль:** {что болит прямо сейчас}
- **Хроническая боль:** {что тянется годами}
- **Скрытая боль:** {о чём не говорят вслух}
- **Триггерные события:** {что запускает поиск решения}

### 🚫 Возражения
- **«Дорого»** — почему не купит за эту цену
- **«Нет времени»** — какой барьер времени
- **«Не верит»** — сомнения в результате/авторе
- **«Сделаю сама»** — почему не нужен продукт
- **«Подумаю»** — реальная причина откладывания

### 🚪 Точки входа
- **Где ищет информацию:** {каналы}
- **Где общается с «такими же»:** {сообщества}
- **Где покупает похожее:** {маркетплейсы, магазины, инфобиз}
- **Кому доверяет:** {лидеры мнений, формат доверия}

### 🔍 Ключевые запросы
- **Прямые:** {что гуглит/ищет в ОК/Дзен}
- **Косвенные:** {что ищет, не зная решения}
- **Соседние темы:** {что ещё изучает параллельно}

### 💎 Главный инсайт
{1 предложение — главное, что нужно понять про этот сегмент}
```

## Тон

- **Честный анти-маркетинг**: никаких идеализаций
- **Конкретика**: цифры, примеры, цитаты (если есть исследования)
- **Без воды**: каждый пункт = действие для маркетинга

## Пример

```markdown
## 👤 ПОРТРЕТ СЕГМЕНТА B: «На перекрёстке»

### 📊 Демография
- **Пол:** женский (95%)
- **Возраст:** 35-44 года
- **Гео:** Россия, города 100к+, в основном ЦФО/ПФО/СЗФО
- **Доход:** 40-90к ₽/мес на семью, часто единственный стабильный
- **Семейное положение:** замужем (60%) / в разводе (20%) / не замужем (20%)
- **Образование:** высшее (70%)
- **Занятость:** офис/удалёнка/свой мини-бизнес, часто совмещает с детьми

### 💔 Боли
- «Всё для всех, а для себя — ничего»
- «Муж как сосед, дети подросли, а я не знаю, что дальше»
- «Кризис среднего возраста, но никому не скажешь»
- «Устала без причины, врачи говорят «здорова», а внутри — пусто»
- «Когда-то мечтала о другой жизни, а теперь — будни»

### 🚪 Точки входа
- **ОК:** группы «Женский взгляд», «Жизнь после 35», «Психология для нас»
- **Дзен:** статьи про кризис среднего возраста, исполнение желаний
- **VK:** сообщества по саморазвитию, психологии, мамству
- **Покупает:** курсы на GetCourse (200-1000 ₽), книги на Литрес, вебинары

### 💎 Главный инсайт
**Она не ищет «волшебную таблетку» — она ищет подтверждение, что жизнь может быть другой, и инструкцию, как это сделать, не разрушив семью и не потеряв деньги.**
```
