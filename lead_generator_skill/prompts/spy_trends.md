# /lead spy-trends — тренды ниши

## Что делает

Находит **топ креативов ниши** (не конкретного конкурента, а всех в категории) и выделяет:
- Самые частотные офферы
- Самые «крутящиеся» заголовки (те, что давно в ротации)
- Популярные триггеры (страх, желание, любопытство)
- Тон, который доминирует

## Источники

| Источник | Что даёт | Скрипт |
|----------|----------|--------|
| **ads.vk.com** | Все активные креативы ниши (поиск по ключу) | `spy_vk_ads.py` |

Дополнительно вручную:
- Топ-посты в VK-пабликах ниши
- Популярные заголовки в Дзен
- Темы в TG-каналах

## Формат ответа

```markdown
# 📈 Тренды ниши: [ниша]

## 1. Топ-5 офферов (по частотности)
1. «Триал 7 дней бесплатно» — 40% креативов
2. «PDF-гайд бесплатно» — 35%
3. «Скидка 50% первым 100» — 20%
4. «Бесплатная консультация 15 мин» — 15%
5. «Гарантия возврата 30 дней» — 10%

## 2. Топ-5 заголовков
1. «[Боль/ситуация] — что делать» (шаблон)
2. «Как [результат] за [срок] без [ограничение]»
3. «[Число] [кейсов/историй] о [тема]»
4. «Не верю в [что-то], но...»
5. «[Возраст/ситуация] — почему [проблема]»

## 3. Популярные триггеры
- **Страх:** «упустишь время», «потом поздно»
- **Желание:** «наконец-то для себя», «перестать жить на автопилоте»
- **Любопытство:** «почему не работает X», «секрет Y»
- **Социальное доказательство:** «[число] женщин уже...»

## 4. Доминирующий тон
- 60% — рациональный («5 шагов», «пошаговый PDF»)
- 30% — эмоциональный («истории», «от первого лица»)
- 10% — анти-маркетинг («не верю, но...»)

**Возможность:** анти-маркетинг **недозанят** (10% vs 30-60% рационала). Можно выделиться.

## 5. Каналы (где крутят)
| Канал | Доля | CPL (примерно) |
|-------|------|----------------|
| ОК Реклама | 30% | 400-1000 ₽ |
| VK Реклама | 25% | 600-1500 ₽ |
| Я.Директ | 20% | 800-2000 ₽ |
| TG-посевы | 15% | 400-800 ₽ |
| Дзен | 10% | 300-700 ₽ |

**Возможность:** Дзен (самый дешёвый) **недозанят** нишей. Можно зайти.

## 6. Рекомендация

**Наш оффер:** триал 7 дней + PDF-магнит «7 шагов» (как у топов).
**Наш тон:** анти-маркетинг (как у 10%, но с лучшей подачей).
**Наш канал-фокус:** ОК + Дзен (недозаняты + дешёвые).
```

## Запуск

```bash
# Найти 50 креативов ниши в VK Ads
PYTHONIOENCODING=utf-8 python scripts/spy_vk_ads.py --query "[ниша]" --limit 50 --output trends.csv
```

## Что делает скил после

1. Запускает парсер
2. Считывает частоты (слова, офферы, триггеры)
3. Группирует по категориям
4. Сравнивает с бенчмарками из `data/benchmarks_ru_2026.md`
5. Выдаёт тренд-отчёт + 3 рекомендации

## Когда использовать

- **При запуске новой ниши** (быстро понять ландшафт)
- **Каждые 3 месяца** (тренды меняются)
- **При выборе оффера** (что реально работает, а не «мне кажется»)
- **Для вдохновения креативов** (но не копирования!)

## Связь с wordstat

```bash
# 1) Спрос (wordstat)
/lead wordstat drafts/<ниша>.md

# 2) Конкурентный анализ трендов (эта команда)
/lead spy-trends <ниша>

# 3) Боли конкурентов
/lead spy-reviews <конкурент>
```

**Зачем связка:** wordstat показывает **что люди ищут** (цифры спроса), spy-trends — **что конкуренты крутят** (предложение). Если спрос большой, а предложение не закрывает — это **окно возможностей**.

> **Пример:** wordstat «карта желаний» = 46 838 (но 80% — про ТВ-сериал) + spy-trends по нише «коучинг по визуализации» = мало конкурентов. → Окно: писать в Дзен под коучинговый интент (а не под развлекательный).
