# /lead spy — разведка по одному конкуренту

## Что делает

Собирает из открытых источников:
1. Какие каналы использует конкурент (VK, ОК, TG, Дзен, Я.Директ)
2. Какие креативы крутит (заголовки, тексты, тона)
3. Какие офферы предлагает (триал, пакет, цена)
4. Какие отзывы получает (плюсы, минусы, боли клиентов)

## Источники (легально, бесплатно)

| Источник | Что даёт | Скрипт |
|----------|----------|--------|
| **ads.vk.com** | Креативы из VK Ads Library (ОК + VK) | `spy_vk_ads.py` |
| **tgstat.com** | Топ TG-каналов в нише + статистика | `spy_tg_channels.py` |
| **otzovik.com** | Отзывы клиентов + плюсы/минусы | `spy_reviews.py` |
| **Поиск руками** | Сайт конкурента, Дзен-канал, VK-паблик | — |

## Формат ответа

```markdown
# 🕵️ Spy-отчёт: [Конкурент / Бренд]

## 1. Кто это
- Название, ниша, ЦА
- Сколько лет на рынке
- Известен по чему

## 2. Каналы (что использует)
| Канал | Активность | Связь с ЦА |
|-------|-----------|------------|
| VK Реклама | 🟢 активно | сегмент A, B |
| ОК Реклама | 🟡 слабо | сегмент C, D |
| Telegram | 🟢 активно | все |
| Дзен | 🟡 статьи | B, C |
| Я.Директ | 🟢 поиск | горячий спрос |

## 3. Креативы (топ-5)
1. **Заголовок:** ...
   **Текст:** ...
   **Оффер:** ...
   **Тон:** рациональный / эмоциональный / анти-маркетинг
2. ...

## 4. Офферы
- Триал: 7 дней бесплатно → 590 ₽/мес
- Пакет: 3 мес за 1490 ₽ (скидка 15%)
- Апсейл: индивидуальный коучинг 4900 ₽

## 5. Боли клиентов (из отзывов)
- «Дорого» (12×)
- «Не работает» (8×)
- «Много воды» (6×)
- «Не верю в эзотерику» (5×)

## 6. Что взять / что улучшить

**Взять (что у конкурента работает):**
- Триал-механика
- PDF-магнит «7 шагов»
- Честные истории с цифрами

**Улучшить (где конкурент проваливается):**
- Слабый TG-канал (можно обойти)
- Нет ОК-рекламы (огромный сегмент 45+)
- Шаблонные креативы (наша anti-эстетика выиграет)

## 7. Рекомендация

Запускать ли кампанию против конкурента? Если да — на чём строить дифференциацию.
```

## Запуск скриптов

```bash
# Креативы конкурента в VK Ads
PYTHONIOENCODING=utf-8 python scripts/spy_vk_ads.py --query "[имя конкурента]" --limit 30 --output spy_vk.csv

# Топ TG-каналов в нише
PYTHONIOENCODING=utf-8 python scripts/spy_tg_channels.py --query "[ниша]" --min-subs 3000 --output spy_tg.csv

# Отзывы
PYTHONIOENCODING=utf-8 python scripts/spy_reviews.py --query "[продукт конкурента]" --site otzovik --limit 30 --output spy_reviews.csv
```

## Что делает скил после запуска

1. Запускает 3 скрипта параллельно (если указаны)
2. Читает CSV-файлы
3. Делает частотный анализ офферов, болей, каналов
4. Генерирует spy-отчёт по формату выше
5. Подсвечивает 3-5 «окон возможностей» (где конкурент слаб)

## Когда использовать

- Перед запуском новой ниши (понимание ландшафта)
- Перед масштабированием (ищем что ещё не тестировал конкурент)
- Каждые 2-3 месяца (мониторинг изменений)
- При выборе оффера (что работает у других)

## Связь с wordstat и другими командами

```bash
# Полный цикл: спрос → что крутят → боли → оффер
/lead wordstat drafts/<ниша>.md       # 1) что ищут люди
/lead spy-trends <ниша>                # 2) как таргетят конкуренты
/lead spy <конкурент>                  # 3) полный профиль конкурента
/lead spy-reviews <продукт>            # 4) боли и цитаты
/lead offer <сегмент>                  # 5) наш оффер
```

> **Лайфхак:** для одной ниши wordstat + 3 spy-команды дают полную картину: что люди ищут + как конкуренты таргетят + где у них болит + как это закрыть.
