# 📡 Промпт: Стратегия по каналам

## Назначение
Команда `/lead стратегия [ниша] [бюджет] [цель]`. Общая стратегия продвижения по каналам.

## Подход к РФ-рынку 2026

1. **ОК + Дзен = основа** для ЦА 35-55 (дешёвые лиды)
2. **VK Реклама = усиление** для ЦА 25-44
3. **Telegram** = точечный (посевы в каналах, не Ads)
4. **Я.Директ** = горячий спрос (поиск)
5. **Авито/Вебинары** = нишевые
6. **Yappy/Threads** = тест 5-10% бюджета

## Алгоритм

1. Определи **ЦА** (4+ сегмента)
2. Определи **продукт, чек, цикл сделки**
3. Рассчитай **допустимый CAC** от LTV
4. Распредели **бюджет по каналам** с приоритетами
5. Сделай **план по месяцам** (тест → масштаб → оптимизация)
6. Дай **KPI по каждому каналу**

## Формат ответа

```markdown
## 📊 СТРАТЕГИЯ ПРОДВИЖЕНИЯ

### 1. Вводные
- **Ниша:** {ниша}
- **Продукт:** {что продаём}
- **Чек:** {цена}
- **Цикл сделки:** {срок}
- **Целевой сегмент:** {A/B/C/D}
- **Бюджет:** {X} ₽/мес
- **Цель:** {что хотим достичь}

### 2. Экономика
- **Целевая выручка:** {X} ₽/мес
- **LTV клиента:** {X} ₽
- **Допустимый CAC:** {X} ₽
- **Минимальный ROI:** {X}x

### 3. Каналы и бюджет

| Канал | % бюджета | Сумма/мес | Ожидаемый результат | CPL |
|-------|-----------|-----------|---------------------|-----|
| ОК Реклама | 35% | X ₽ | Y лидов, Z подписок | X ₽ |
| Дзен (продвижение) | 20% | X ₽ | Y лидов, Z подписок | X ₽ |
| VK Реклама | 20% | X ₽ | Y лидов, Z подписок | X ₽ |
| Посевы TG | 10% | X ₽ | Y лидов, Z подписок | X ₽ |
| Я.Директ (РСЯ) | 10% | X ₽ | Y лидов, Z подписок | X ₽ |
| Авито (тест) | 5% | X ₽ | Y лидов, Z подписок | X ₽ |

### 4. План по месяцам

**Месяц 1 — тест (10-15% бюджета)**
- Запустить по 2-3 креатива в каждом канале
- Цель: найти связку канал-креатив с лучшим CPL
- Собрать базу 200+ контактов
- KPI: CPL не выше {X} ₽

**Месяц 2 — масштабирование (50-60% бюджета)**
- Увеличить бюджет на лучшую связку ×2
- Подключить дополнительные креативы
- Запустить ретаргет
- KPI: выйти на {X} подписок/мес

**Месяц 3 — оптимизация (полный бюджет)**
- Отключить слабые связки
- Масштабировать лучшие
- Запустить апсейл/рефералку
- KPI: {цель}

### 5. Подготовка (до старта)
- [ ] Лендинг с формой (скорость < 2 сек, мобильный)
- [ ] Лид-магнит (PDF или видео)
- [ ] Бот для выдачи (BotHelp/Senler)
- [ ] CRM для учёта (amoCRM)
- [ ] Рекламные кабинеты (ОК/VK/Я.Директ)
- [ ] Маркировка ОРД
- [ ] Политика ПД + форма согласия
- [ ] ЮKassa/Т-Касса + чеки

### 6. Риски и контроль
- **Если CPL > X ₽** через 3 дня → отключить креатив
- **Если подписки < 5% от лидов** → проблема в лендинге/прогреве
- **Если ROI < 1.5x** → пересмотреть каналы
```

## Пример для Лаборатории желаний (Сегмент B, 50к бюджет)

```markdown
## 📊 СТРАТЕГИЯ: «Подписка Лаборатория желаний» (Сегмент B 35-44)

### 1. Вводные
- Чек: 590 ₽/мес, цикл: 6 мес LTV = 3540 ₽
- Допустимый CAC: 1200 ₽
- Бюджет: 50 000 ₽/мес
- Цель: 50 подписок/мес

### 3. Каналы
- ОК Реклама: 20 000 ₽ → 30-40 лидов, 15-20 подписок (CPL 500 ₽)
- Дзен (продвижение): 10 000 ₽ → 20-30 лидов, 8-12 подписок (CPL 350 ₽)
- VK Реклама: 10 000 ₽ → 15-25 лидов, 7-10 подписок (CPL 600 ₽)
- Посевы TG: 5 000 ₽ → 8-12 лидов, 3-5 подписок
- Я.Директ (РСЯ): 5 000 ₽ → 8-12 лидов, 3-5 подписок
- **ИТОГО:** 50 000 ₽ → 80-120 лидов → **35-50 подписок**

### 4. План
- Мес 1 (тест 30к): найти лучшую связку, CPL не выше 700 ₽
- Мес 2 (масштаб 50к): ×2 лучший канал, ретаргет
- Мес 3 (оптимизация 50к): отключить плохое, добавить вебинар
```

## Тон

Стратегия — это не «давайте попробуем». Это **конкретный план с цифрами, KPI, дедлайнами**. Без воды.
