# 🔗 Интеграция: WishLibrarian (генерация PDF-лид-магнита)

## Назначение
Команда `/lead magnet-from-wl [книга]`. Превращает книгу из WishLibrarian в PDF-лид-магнит.

## Что это
Автоматический пайплайн: **книга → конспект → PDF-магнит → лендинг → цепочка писем**.

## Шаги

### 1. Получить книгу из WishLibrarian
```bash
# В WishLibrarian:
/process book "Ты в порядке" author "Наталья Кудряшова"
# → Готовый конспект в JSON + Markdown
```

### 2. LeadGen преобразует в PDF
- **Титульная страница** (обложка через `cover-generator` или Canva)
- **Лид-абзац** (боль → обещание → CTA, 200 слов)
- **5-7 глав** из конспекта (каждая 2-3 стр)
- **Практические задания** (после каждой главы)
- **P.S.** с призывом к подписке
- **Дисклеймер** (для ниши психологии/саморазвития)
- **Контакты автора**

### 3. Создать лендинг через Publisher
- **Структура:** заголовок → описание → отзывы → форма → footer
- **Форма:** имя + email
- **Кнопка:** «Скачать PDF бесплатно»
- **Скорость:** < 2 сек
- **Мобильный:** адаптивный
- **Маркировка:** если реклама — добавить ERID

### 4. Подключить цепочку писем
- **Email-сервис:** Sendpulse или UniSender
- **Цепочка:** 5 писем за 7 дней
- **Темы:**
  - День 0: «Спасибо! Вот твой PDF» (мгновенно)
  - День 1: «Как применить быстрее» (3 лайфхака)
  - День 2: «История Марины, 38 лет» (кейс)
  - День 3: «Что ещё почитать» (рекомендации)
  - День 5: «Твой персональный план» (триал)

### 5. Запустить трафик
- **Каналы:** ОК, Дзен, VK, посевы TG
- **Креативы:** «PDF-гайд на основе книги [название]»
- **CPL магнита:** 40-150 ₽
- **CPL подписки:** 500-1200 ₽

## Шаблон команды

```bash
/lead magnet-from-wl "Ты в порядке" "Наталья Кудряшова"
  → Скил генерирует:
     1. Структуру PDF (титул + 5 глав + задания + P.S.)
     2. Текст каждой главы (на основе конспекта WL)
     3. Лид-абзац (боль → обещание → CTA)
     4. Тему обложки (для `cover-generator`)
     5. Структуру лендинга (для Publisher)
     6. Цепочку из 5 писем
     7. 3 креатива для трафика
     8. Прогноз CPL/CAC
```

## Мониторинг

- **Скачивания:** Яндекс.Метрика (цель «скачал PDF»)
- **Подписки:** CRM (цель «оплатил»)
- **Конверсия:** 5-15% (магнит → подписка)
- **CPL:** отслеживать по каналам

## Требования к книге

✅ Книга есть в WishLibrarian
✅ Конспект сделан
✅ Автор дал согласие (или книга в public domain)
✅ Конспект > 20 страниц (для PDF-магнита)
✅ Есть практические задания (для активного применения)

## Частые ошибки

❌ Слишком короткий PDF (< 5 стр) — низкая конверсия
❌ Просто пересказ книги — нет ценности
❌ Нет моста к подписке — теряются лиды
❌ Нет дисклеймера — риски ФАС
