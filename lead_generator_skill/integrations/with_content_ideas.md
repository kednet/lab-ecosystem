# 🔗 Интеграция: Content Ideas (идеи постов)

## Назначение
Использует Content Ideas Skill для генерации идей постов без LLM-токенов. Применяется в `/lead креативы`, `/lead dzen`, `/lead webinar`.

## Что это
Бесплатная (без LLM-токенов) генерация идей контента на основе анализа конкурентов, трендов, существующих книг.

## Зачем

- **LLM-токены стоят денег** (особенно для больших объёмов)
- **Content Ideas** работает на rule-based + парсинге (0 ₽)
- **Подходит для:** регулярного контента, прогрева, постов в ОК/Дзен/TG

## Шаги

### 1. Запуск Content Ideas
```bash
# В Content Ideas:
/ci suggest "карта желаний" --limit 30
  → 30 идей постов

/ci ok "как вернуть себя" --count 10
  → 10 идей для ОК
```

### 2. LeadGen использует идеи
- **Подбор формата** (ОК-пост, Дзен-статья, TG-пост)
- **Адаптация под тон** (честный анти-маркетинг)
- **Структура** (крючок → ситуация → перелом → совет → CTA → P.S.)
- **Длина** (800-1500 знаков для ОК, 2000-3000 для Дзен)

### 3. Доработка
- **Добавить** конкретику (цифры, имена, истории)
- **Добавить** дисклеймер (для ниши психологии)
- **Добавить** CTA (к лид-магниту/подписке)
- **Проверить** стоп-слова ФАС

## Шаблон команды

```bash
# Генерация постов для прогрева
/lead прогрев "карта желаний" --platforms ok,dzen,tg --count 10
  → Скил:
     1. Запускает Content Ideas
     2. Берёт топ-10 идей
     3. Адаптирует под каждый канал
     4. Добавляет структуру
     5. Проверяет тон
     6. Возвращает готовые посты
```

## Что берёт Content Ideas

- **Темы:** из банка идей (29+ идей для саморазвития)
- **Структуры:** 5-7 шаблонов постов
- **Ключевые слова:** из Wordstat, TGStat
- **Тренды:** из VK, ОК, Дзен
- **Конкуренты:** парсинг ТОП-постов

## Что добавляет LeadGen

- **Тон:** честный анти-маркетинг
- **Структура:** 8 блоков (заголовок → ситуация → история → перелом → совет → CTA → P.S.)
- **Конкретика:** цифры, имена, истории
- **Дисклеймер:** для ниши психологии
- **CTA:** к лид-магниту/подписке
- **Длина:** под канал

## Когда использовать

✅ **Регулярный контент** (3-5 постов в неделю)
✅ **Прогрев** перед вебинаром
✅ **Поддержка SEO** (статьи в Дзен)
✅ **Репосты** в разные каналы (адаптация)
✅ **Новые темы** (когда «не знаешь, о чём писать»)

❌ **Когда нужен уникальный контент** (важные запуски, продуктовые посты)
❌ **Когда нужна экспертная позиция** (важные темы — лучше вручную)

## Главный плюс

> **0 ₽ за генерацию контента.** Content Ideas работает на rule-based + парсинге. Не тратит LLM-токены. Это критично для регулярного контента.

## Пример workflow

```bash
# Понедельник: 5 идей для ОК
/lead прогрев "как вернуть себя" --platform ok --count 5

# Вторник: 1 статья для Дзен
/lead dzen "почему карты желаний не работают"

# Среда: 3 поста для TG
/lead прогрев "кризис среднего возраста" --platform tg --count 3

# Четверг: 1 пост для VK
/lead креативы vk "PDF-гайд" 5000

# Пятница: 1 видео-сценарий для клипа
/lead креативы yappy "5 фраз утром" 1000
```

## Метрики

- **Количество постов:** 3-7 в неделю (зависит от каналов)
- **Время на 1 пост:** 5-10 мин (после первых 3-5 — быстрее)
- **Стоимость:** 0 ₽ (Content Ideas) или 5-10 ₽ (LLM)
- **Конверсия в подписку:** 0.5-2% (с канала)
