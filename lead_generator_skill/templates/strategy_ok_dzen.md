# 📊 Шаблон: Стратегия «ОК + Дзен» (основа)

## Назначение
Базовый шаблон канальной стратегии, где **ОК + Дзен = 50-60% бюджета**, остальное — усиление.

## Почему ОК + Дзен — лучшая связка для ЦА 35-55

| Параметр | ОК | Дзен |
|----------|-----|------|
| MAU | 36 млн | 70 млн |
| Ядро ЦА | ж 35-55 | ж/м 30-55 |
| CPL (магнит) | 40-120 ₽ | 0-50 ₽ (органика) / 80-200 ₽ (продвижение) |
| CPL (подписка) | 400-1000 ₽ | 300-700 ₽ |
| Формат | Пост 800-1500 знаков, видеоклипы 9:16 | Статья 2000-3500 знаков |
| Тон | Тёплый, длинный, без FOMO | Экспертный, спокойный |
| Сезонность | Слабая (просадка летом) | Слабая |
| Главный плюс | Доверие мам/«своих» | SEO-эффект, долгоиграющие статьи |

**Комбо:** статья Дзен прогревает → пост ОК дожимает → подписка.

## Шаблон стратегии

```markdown
## 📊 СТРАТЕГИЯ «ОК + ДЗЕН» ДЛЯ {продукт}

### 1. Вводные
- **Продукт:** {название, чек, формат}
- **Целевой сегмент:** {A/B/C/D}
- **Бюджет:** {X} ₽/мес
- **Цель:** {Y новых клиентов/мес}
- **Допустимый CAC:** {Z} ₽ (от LTV)

### 2. Экономика
- **LTV клиента:** {средний чек × срок подписки × маржа}
- **Допустимый CAC:** {LTV / 3} (соотношение 3:1)
- **Минимальный ROI:** 1.5x (CAC / LTV)

### 3. Распределение бюджета

| Канал | % | Сумма/мес | Роль | Ожидаемый CPL |
|-------|---|-----------|------|---------------|
| **ОК Реклама** | 30% | X ₽ | Основа | 500 ₽ |
| **Дзен (продвижение)** | 20% | X ₽ | Основа | 350 ₽ |
| VK Реклама | 20% | X ₽ | Усиление | 600 ₽ |
| Посевы TG | 10% | X ₽ | Усиление | 700 ₽ |
| Я.Директ (РСЯ) | 10% | X ₽ | Усиление | 500 ₽ |
| Авито (тест) | 5% | X ₽ | Нишевый | 1000 ₽ |
| Вебинар (1 раз) | 5% | X ₽ | Дожим | 1500 ₽ |

### 4. Креативы по каналам

#### ОК Реклама (3-5 креативов)
1. История-отзыв (1200 знаков + фото)
2. Анти-маркетинг («Не верю в X...»)
3. UGC (скриншот переписки)
4. Видеоклип 9:16 (30 сек, субтитры)
5. Длинный пост-разбор (1500 знаков)

#### Дзен (2-3 статьи в месяц)
1. «Почему X не работает (и что делать)» — 2500 знаков
2. «5 шагов к Y» — 2500 знаков
3. «3 истории наших подписчиц» — 3000 знаков

#### VK Реклама (3-5 креативов)
1. Эмоциональный
2. Рациональный
3. Анти-маркетинговый
4. Видео (15 сек)
5. Карусель (5 карточек)

#### Посевы TG (2-3 посева в микро-каналах)
- Тематика: саморазвитие, психология, женское
- Подписчики: 5-30к
- Цена: 1500-3000 ₽
- Формат: пост 500-800 знаков + фото

#### Я.Директ (5-10 объявлений)
- Поиск: горячие ключи
- РСЯ: ретаргет + look-alike

#### Авито (1-3 объявления)
- Категория: Услуги → Обучение
- Формат: консультация

### 5. План по месяцам

**Месяц 1 — тест (60% бюджета, 30 000 ₽)**
- Запустить 2-3 креатива в ОК, 1 статью в Дзен, 1 посев TG
- Цель: найти связку канал-креатив с лучшим CPL
- Собрать базу 100+ контактов
- KPI: CPL < 700 ₽

**Месяц 2 — масштаб (100% бюджета, 50 000 ₽)**
- Увеличить бюджет на лучшую связку ×2
- Добавить новые креативы
- Запустить ретаргет на не-конвертировавших
- KPI: {X} подписок/мес

**Месяц 3 — оптимизация (100% бюджета)**
- Отключить слабые связки
- Добавить вебинар (1 раз в месяц)
- Запустить апсейл/рефералку
- KPI: выйти на целевую юнит-экономику

### 6. Синергия каналов

```
ОК (холодный) → Статья Дзен (прогрев) → Лид-магнит (PDF) → 
→ Цепочка 5 писем → Триал 7 дней → Подписка → Апсейл
```

**Пример:**
1. Девушка видит пост ОК → переходит на лендинг
2. Скачивает PDF «7 шагов» (лид-магнит)
3. Получает цепочку из 5 писем за 7 дней
4. День 5: триал 7 дней бесплатно
5. День 12: оффер на подписку
6. День 19: апсейл (пакет «3 месяца»)

### 7. Метрики для контроля

| Метрика | Целевое значение | Действие при отклонении |
|---------|------------------|--------------------------|
| CPL ОК | 40-120 ₽ | > 200: сменить креатив |
| CPL Дзен | 0-50 ₽ | > 100: сменить тему |
| CPL VK | 80-250 ₽ | > 400: сменить сегмент |
| Конверсия лендинг → магнит | 10-20% | < 5%: переделать лендинг |
| Конверсия магнит → триал | 5-15% | < 3%: переделать цепочку |
| Конверсия триал → подписка | 20-40% | < 15%: переделать оффер |
| ROI | > 1.5x | < 1.0: резать каналы |

### 8. Подготовка до старта
- [ ] Лендинг (через Publisher)
- [ ] Лид-магнит (PDF через WL или Canva)
- [ ] Бот для выдачи (BotHelp/Senler)
- [ ] CRM (amoCRM)

- [ ] Метрика с целями
- [ ] UTM-метки
```

## Топ-комбинации ОК+Дзен

| Связка | Когда использовать | CPL |
|--------|---------------------|-----|
| **ОК-пост → Лендинг → Магнит → Цепочка** | Основная воронка | 400-700 ₽ |
| **Дзен-статья → Лендинг → Магнит → Цепочка** | Холодный трафик, прогрев | 300-500 ₽ |
| **Дзен-статья → OK Live (эфир) → Триал → Подписка** | Горячий трафик | 500-800 ₽ |
| **ОК-видеоклип → Бот в TG → Цепочка → Триал** | Мобильная аудитория | 350-600 ₽ |
| **ОК-пост + Дзен-статья (одна тема) → Лендинг** | Усиление | 250-500 ₽ |

## Почему нельзя опираться только на VK

- VK Реклама: CPL 600-1500 ₽ (дороже)
- ЦА 35-55: меньше представлена в VK, чем в ОК
- Дзен: SEO-эффект, статьи живут месяцами
- ОК: доверие + длинный формат
- VK: нужен, но не основа
