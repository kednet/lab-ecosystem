# 🎯 Шаблон: Чек-лист лендинга

## Назначение
Команда `/lead landing-check`. 47-пунктовый чек-лист для лендинга, оптимизированного под РФ-2026.

## 1. Скорость и технические параметры (P0)

- [ ] Загрузка < 2 секунды (PageSpeed Insights > 80 на мобильных)
- [ ] First Contentful Paint < 1.5 сек
- [ ] Largest Contentful Paint < 2.5 сек
- [ ] Cumulative Layout Shift < 0.1
- [ ] Размер страницы < 1 МБ (без видео)
- [ ] Сжатие Gzip/Brotli включено
- [ ] Изображения в WebP/AVIF
- [ ] Lazy-load для изображений и видео
- [ ] CDN настроен
- [ ] SSL-сертификат валиден
- [ ] Мобильная версия (75%+ трафика в РФ)
- [ ] AMP-версия (опционально, для рекламы)

## 2. SEO-параметры (через SEO Advisor)

- [ ] Title с ключом (60 символов)
- [ ] Meta description с ключом и CTA (155 символов)
- [ ] H1 = 1, с ключом
- [ ] H2-H3 структура логичная
- [ ] Alt-теги на изображениях
- [ ] Schema.org разметка (Article, Product, FAQPage)
- [ ] Open Graph (для репостов)
- [ ] Canonical URL
- [ ] Sitemap.xml обновлён
- [ ] Robots.txt не блокирует индексацию

## 3. Контент первого экрана (P0)

- [ ] Заголовок виден без скролла
- [ ] Заголовок отвечает на 3 вопроса за 5 секунд:
  - Что это?
  - Для кого?
  - Что я получу?
- [ ] Подзаголовок усиливает заголовок (1-2 строки)
- [ ] Изображение/видео первого экрана (лицо, продукт, ситуация)
- [ ] CTA-кнопка видна без скролла
- [ ] Текст кнопки конкретный: «Скачать PDF», «Начать бесплатно», «Получить гайд»
- [ ] Цвет кнопки контрастный к фону
- [ ] Размер кнопки удобен для тапа пальцем (min 44×44 px)

## 4. Социальное доказательство (P1)

- [ ] 3-5 отзывов реальных клиентов (с фото, именем, возрастом)
- [ ] Числа: «Уже 500+ подписчиц», «За 2 года — 1000+ историй»
- [ ] Логотипы СМИ, где вас упоминали (если есть)
- [ ] Скриншоты переписок (с разрешения)
- [ ] Видео-отзыв (30-60 сек) на главной
- [ ] Рейтинги (LiveLib, Литрес, Яндекс.Карты)

## 5. Форма сбора контакта (P0)

- [ ] Минимум полей: только email (или только телефон)
- [ ] Если несколько полей — мотивация: «Имя — чтобы обращаться лично»
- [ ] Галочка согласия на обработку ПД (НЕ пречек)
- [ ] Ссылка на политику ПД (рядом с галочкой)
- [ ] Текст кнопки: «Скачать», «Получить», «Подписаться» (не «Отправить»)
- [ ] Валидация email/телефона
- [ ] Сообщение об успехе после отправки
- [ ] Редирект на thank-you страницу (с ссылкой на магнит)
- [ ] Интеграция с email-сервисом (Sendpulse/UniSender)
- [ ] Интеграция с CRM (amoCRM/Bitrix24)
- [ ] UTM-метки сохраняются

## 6. Юридический блок (P0)

- [ ] Политика обработки ПД (ссылка в футере)
- [ ] Пользовательское соглашение / Оферта
- [ ] Cookies-баннер (если используешь Метрику)
- [ ] Дисклеймер (для ниши психологии/саморазвития)
- [ ] Реквизиты (ФИО, ИНН для НПД) — в футере
- [ ] Контактные данные (email для обратной связи)

## 7. Доверие (P1)

- [ ] Фото автора/команды (реальные, не стоковые)
- [ ] История автора: «Кто я, почему мне можно верить»
- [ ] Образование, опыт, сертификаты (если есть)
- [ ] Контакты для связи (email, TG, телефон)
- [ ] Социальные сети (ссылки на VK, TG, ОК)
- [ ] Юридический адрес / ИНН (в оферте)

## 8. Контентные блоки (P2)

- [ ] Блок «О продукте» (что входит, как работает)
- [ ] Блок «Для кого» (под какие сегменты/боли)
- [ ] Блок «Как это работает» (3-5 шагов)
- [ ] Блок «Тарифы» (если несколько)
- [ ] Блок «Отзывы» (расширенный)
- [ ] Блок «FAQ» (5-7 вопросов с ответами)
- [ ] Блок «Гарантия возврата» (если есть)
- [ ] Финальный CTA (повтор)

## 9. Мобильная версия (P0)

- [ ] Адаптивный дизайн (все элементы видны)
- [ ] Текст читаемый (min 16px)
- [ ] Кнопки крупные (min 44×44 px)
- [ ] Формы простые (без капчи или простая)
- [ ] Видео не автоплеит (раздражает)
- [ ] Нет горизонтального скролла
- [ ] Sticky-кнопка CTA (всегда видна)
- [ ] Меню-гамбургер (если есть меню)
- [ ] Свайп-жесты работают (если есть карусели)
- [ ] Картинки сжаты для мобильных

## 10. Аналитика (P0)

- [ ] Яндекс.Метрика установлена
- [ ] Цели настроены: отправка формы, клик на кнопку, скролл 50/100%
- [ ] Вебвизор включён (для анализа поведения)
- [ ] Google Analytics 4 (опционально, для SEO)
- [ ] UTM-метки читаются
- [ ] Пиксели рекламных кабинетов (VK, ОК, Я.Директ)
- [ ] UTM-разметка для всех ссылок

## 11. A/B-тесты (P2)

- [ ] Тест заголовка (минимум 2 варианта)
- [ ] Тест CTA-кнопки (цвет, текст, размер)
- [ ] Тест изображения (лицо vs продукт)
- [ ] Тест формы (только email vs email+имя)
- [ ] Инструмент: Google Optimize (или A/B через рекламный кабинет)

## 12. Финальная проверка (перед запуском)

- [ ] Просмотр на 3 устройствах (iPhone, Android, десктоп)
- [ ] Просмотр в 3 браузерах (Chrome, Safari, Firefox)
- [ ] Тест скорости: PageSpeed, GTmetrix
- [ ] Тест форм: отправка работает?
- [ ] Тест оплаты: демо-режим
- [ ] Тест на ошибки: 404, 500
- [ ] Бэкап предыдущей версии

## 13. Конверсионные триггеры (P1)

- [ ] Таймер обратного отсчёта (если есть дедлайн)
- [ ] «Осталось 5 мест» (если реально)
- [ ] «Сегодня бесплатно» (реальный дедлайн)
- [ ] Счётчик подписчиков (если есть)
- [ ] Реальные фото/видео (не стоковые)
- [ ] Истории с цифрами («за 30 дней 50 подписчиц»)

## 14. Видео (P2)

- [ ] Видео на лендинге (30-90 сек) — автор представляется, показывает продукт
- [ ] Субтитры на видео
- [ ] Автоплеит отключён (или muted с autoplay)
- [ ] Обложка видео с CTA

## Скоринг лендинга

| Блок | Макс. баллов |
|------|--------------|
| Скорость | 15 |
| Контент первого экрана | 15 |
| Форма | 10 |
| Юр.блок | 10 |
| Мобильная версия | 15 |
| Аналитика | 10 |
| Соц.доказательство | 10 |
| SEO | 5 |
| A/B-тесты | 5 |
| Доп.контент | 5 |
| **ИТОГО** | **100** |

**Цель: 80+** — лендинг готов к платному трафику.
**60-80** — нужно доработать перед масштабированием.
**< 60** — не запускать рекламу, теряете деньги.
