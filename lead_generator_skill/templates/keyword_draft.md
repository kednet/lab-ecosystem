# Шаблон черновика семантического ядра для `/lead wordstat`

## Назначение

Этот шаблон — входной файл для команды `/lead wordstat`. Скил прочитает его, проведёт тебя через авторизацию в Яндексе, пошагово соберёт частотности через Playwright и сгенерирует 4 файла:
- `data/wordstat_<slug>_<date>.md` — таблица частотностей
- `data/wordstat_<slug>_clusters_<date>.md` — сортировка А/Б/В/Г/Д
- `data/wordstat_<slug>_titles_<date>.md` — рекомендации по title/h1
- `data/wordstat_<slug>_plan_<date>.md` — контент-план

## Готовый пример

См. `data/draft_lab_zhelanij_keywords.md` (70 фраз по 11 страницам Лаборатории желаний).

## Шаблон

Скопируй этот шаблон, заполни под свой проект, сохрани как `data/draft_<slug>_keywords.md` и запусти `/lead wordstat`.

```markdown
# Семантическое ядро (черновик) — <Название проекта>
## Генерация: <ДД.ММ.ГГГГ>, по N направлениям сайта

---

## 1. <РАЗДЕЛ 1> (`<URL>`) — <описание>

| # | Фраза | Тип интента | Ожид. част. |
|---|---|---|---|
| 1 | <фраза> | <информационный/транзакционный/коммерческий/навигационный> | <ВЫСОКАЯ/СРЕДНЯЯ/НИЗКАЯ> |

## 2. <РАЗДЕЛ 2> (`<URL>`) — <описание>

| # | Фраза | Тип интента | Ожид. част. |
|---|---|---|---|
| 1 | <фраза> | ... | ... |

... (повторить для каждого раздела сайта)

---

## ТОП-N для парсинга через Wordstat

Беру фразы с наибольшей ожидаемой частотностью + коммерческие/транзакционные:

1. <фраза 1>
2. <фраза 2>
...
N. <фраза N>

---

## ПРИМЕЧАНИЕ

Частотность помечена экспертно (без парсинга). После парсинга через `/lead wordstat`
проставлю точные цифры и обновлю оценки.
```

## Правила заполнения

### Что обязательно

- **Структура по разделам сайта** — каждая H2-секция = одна страница/раздел
- **URL в скобках** — для маппинга на конкретные страницы
- **Минимум 2-3 фразы на раздел** — иначе скил выдаст предупреждение «мало данных»
- **ТОП-N в конце** — 20-30 фраз, самые жирные/коммерческие. Именно их скил погонит через Вордстат
- **Тип интента** — обязательно (информационный / транзакционный / коммерческий / навигационный)

### Что опционально

- **Ожидаемая частотность** — экспертная оценка (ВЫСОКАЯ/СРЕДНЯЯ/НИЗКАЯ). Скил после парсинга заменит на точные цифры

### Что НЕ нужно

- **Точные цифры** — их соберёт скил
- **Сортировка по группам А/Б/В/Г/Д** — это делает скил после сбора
- **Рекомендации по title/h1** — это делает скил после сортировки

## Интенты (что выбирать)

| Интент | Когда ставить | Примеры |
|--------|---------------|---------|
| **информационный** | Пользователь ищет знание, инструкцию, объяснение | «как достичь цели», «что такое осознанность», «почему карта желаний не работает» |
| **транзакционный** | Пользователь готов совершить действие (скачать, купить, записаться) | «трекер привычек онлайн», «конспект книги скачать», «записаться на вебинар» |
| **коммерческий** | Пользователь сравнивает цены/предложения, готов купить | «цена подписки», «лучший тариф», «сколько стоит» |
| **навигационный** | Пользователь ищет конкретный сайт/бренд/раздел | «лаборатория желаний», «вход в личный кабинет», «о проекте» |

## Размер черновика

- **Минимум:** 20 фраз в ТОП-N (иначе нечего парсить)
- **Оптимально:** 25-30 фраз в ТОП-N (баланс скорости и полноты)
- **Максимум:** 50 фраз (сбор ~10 минут, не рекомендуется)

## Что делать после сбора

1. Открыть `data/wordstat_<slug>_clusters_<date>.md` — сортировка А/Б/В/Г/Д
2. Открыть `data/wordstat_<slug>_titles_<date>.md` — title/h1 рекомендации
3. Применить рекомендации к своему сайту (вручную или через SEO Advisor)
4. Использовать `data/wordstat_<slug>_plan_<date>.md` для планирования контента

## Связь с другими командами скила

```bash
# Сначала сбор семантики
/lead wordstat data/draft_<slug>_keywords.md

# Потом SEO-страница под топовый запрос
/lead seo-funnel "как достичь цели"

# Или контент-план Дзен
/lead dzen "когнитивные искажения"

# Или креативы для рекламы
/lead креативы ok "медитации онлайн"
```

## Частые ошибки

❌ **Слишком общие фразы** («как жить», «как быть счастливым») — скил не сможет нормально кластеризовать
❌ **Слишком специфичные фразы** (брендовые, с именами) — нулевая частотность
❌ **Фразы без интента** — скил попросит переделать
❌ **ТОП-N совпадает с полным списком** — нет смысла, берите только 25-30 жирных
