# 💰 Шаблон: Объявления для Яндекс.Директ

## Назначение
Команда `/lead креативы yandex [оффер] [бюджет]`. Создаёт готовые объявления для поиска и РСЯ.

## Специфика Яндекс.Директ 2026

- **Поиск:** точно по запросу, высокая цена клика, горячий спрос
- **РСЯ:** баннеры на сайтах-партнёрах, дёшево, холодная/тёплая аудитория
- **Мастер кампаний:** авто-стратегии (максимум кликов/конверсий)
- **Минималка:** 300 ₽ на поиске, 100 ₽ в РСЯ
- **Маркировка:** авто через ОРД + ERID
- **Запреты:** медицина, БАДы, психология — модерация строгая
- **Ниша «психология/саморазвитие»:** нужны дисклеймеры в тексте объявления

## Ограничения символов

| Поле | Поиск | РСЯ |
|------|-------|-----|
| Заголовок 1 | 56 | 56 |
| Заголовок 2 | 30 | 30 |
| Текст | 81 | 81 |
| Отображаемая ссылка | 15 | 15 |
| Быстрые ссылки | 30 (×4) | 30 (×4) |
| Уточнения | 25 (×4) | 25 (×4) |

## Шаблон: Поиск

```markdown

**Заголовок 1 (56):** {Цифра + результат}
**Заголовок 2 (30):** {Снятие возражения}
**Текст (81):** {Выгода + CTA + срок}
**Отображаемая ссылка (15):** {короткий путь}
**Быстрые ссылки (4×30):** {доп. выгоды}
**Уточнения (4×25):** {доп. преимущества}
**Дисклеймер:** Материалы носят информационный характер

### Объявление №2 (ключ + бонус)
...

### Объявление №3 (конкурентное)
...

### 🚀 Рекомендации по запуску
- **Минус-слова:** {список 30-50 минус-слов}
- **Гео:** {города/регионы}
- **Стратегия:** максимум конверсий с CPA = {X} ₽
- **Дневной бюджет:** {X} ₽
- **Ожидаемый CPL:** поиск 200-500 ₽, РСЯ 60-180 ₽
```


## Шаблон: РСЯ

```markdown
## 💰 ОБЪЯВЛЕНИЯ ДЛЯ РСЯ

### Баннер №1 (баннер + текст)
**Заголовок:** {коротко, эмоционально}
**Текст:** {выгода + CTA}
**Изображение:** 1080×607, 600×600 или 240×400
**Видео (опц.):** 10-15 сек, вертикальное

### Креатив в видеоформате
**Сценарий:** 3 кадра — боль → перелом → CTA
**Длительность:** 6-10 сек
**Субтитры:** обязательны
```

## Стоп-слова для модерации Яндекс

❌ «100% результат», «гарантированно», «без риска»
❌ «Лечение», «избавление от», «терапия» (в нише психологии)
❌ «Бесплатно» в заголовке (только в тексте)
❌ Преувеличение («лучший в России», «единственный»)
✅ Дисклеймер обязателен в длинном тексте
