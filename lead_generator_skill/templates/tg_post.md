# 📝 Шаблон: Пост для Telegram (посевы и Ads)

## Назначение
Команда `/lead креативы tg [оффер] [бюджет]`. Создаёт готовые посты для посевов в TG-каналах и нативной рекламы.

## Специфика TG в 2026

- **MAU:** ~85 млн
- **Ядро ЦА:** мужчины/женщины 25-40, IT/маркетинг
- **Формат рекламы:** посевы (ручные/через Telega.in) + официальные TG Ads (через AdRiver/elama, от 1500 €)
- **Длина поста:** 500-800 знаков (оптимально для посевов)
- **Обязательно:** пометка «Реклама» + ERID
- **Топ форматы:** текст + 1 картинка, текст + короткое видео

## Структура поста-посева

1. **Первая строка-крючок** (видна до «читать далее»)
2. **Боль/ситуация** (узнай себя)
3. **Решение** (что предлагаем)
4. **Доказательство** (цифра/отзыв/кейс)
5. **CTA** (что делать)
6. **Маркировка** (обязательно)

## Рекомендации по посевам

- **Выбор каналов:** TGStat → фильтр по тематике «Психология», «Саморазвитие», «Женское», по подписчикам 5-50к
- **Цена поста:** 1000-5000 ₽ (микро), 5000-30000 ₽ (средние), 30000+ ₽ (крупные)
- **Время размещения:** 10:00-12:00, 19:00-22:00 (будни)
- **Формат:** пост от имени канала с пометкой «реклама» (через репост или прямая публикация)
- **Допустимо:** нативная интеграция (без пометки) — только если есть редакционный доступ; иначе обязательно помечать
- **Отслеживание:** UTM-метки + пиксель TG

## Маркировка

**Обязательна** для любой рекламы. Через ОРД (Яндекс.Директ/ВК/Амбер) → токен ERID → добавить в конец поста.
