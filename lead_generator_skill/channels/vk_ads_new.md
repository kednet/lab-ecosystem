# 📺 Канал: VK Реклама (new)

## Обзор
- **MAU:** ~85 млн (с учётом ОК)
- **Ядро ЦА:** мужчины/женщины 18-44, IT/маркетинг/креатив
- **Главный плюс:** мощный таргетинг, look-alike, пиксель
- **Минус:** дороже ОК для ЦА 35-55

## Рекламный кабинет

### Где
- **ads.vk.com** — новый кабинет (с 2023)
- Старый myTarget закрыт
- Старый ВКонтакте Ads закрыт
- **Минимальный бюджет:** 100 ₽/день

### Форматы
1. **Универсальное объявление** (1 креатив → все места)
2. **Видео**
3. **Карусель**
4. **Клипы** (вертикальное видео)
5. **Истории**

### Таргетинг
- Пол, возраст, гео
- **Интересы:** подробные, ~100 категорий
- **Подписчики сообществ:** конкуренты, тематические группы
- **Look-alike:** похожие на подписчиков вашей группы
- **Ретаргет:** по пикселю, по взаимодействию с группой
- **Ключевые фразы** (для поиска)
- **Устройства, ОС, браузеры**

## Что работает в VK

✅ Видеоклипы 9:16 (главный формат)
✅ Универсальные объявления с 1 креативом
✅ Look-alike от 1000+ подписчиков
✅ Ретаргет на скачавших магнит
✅ Таргетинг на конкурентов
✅ Stories + внешние ссылки

## Что НЕ работает в VK

❌ Длинные тексты (сокращаются до 280 на мобильных)
❌ «Глянцевые» стоковые фото
❌ Агрессивный FOMO
❌ Прямые продажи в первом экране

## Ценовой профиль (ниша «саморазвитие/женская»)

- **CPM:** 80-200 ₽
- **CPC:** 8-25 ₽
- **CPL (магнит):** 80-250 ₽
- **CPL (подписка):** 600-1500 ₽
- **CTR:** 1-3%
- **Конверсия в подписку из лид-магнита:** 3-7%

## Стратегия VK Рекламы

### Тестовая связка (месяц 1)
1. **5 креативов** (3 типа: эмоциональный, рациональный, анти-маркетинг)
2. **3 сегмента** (A 25-34, B 35-44, C 45-55)
3. **Look-alike** от подписчиков группы
4. **Бюджет:** 300-500 ₽/день на креатив
5. **Длительность:** 5-7 дней

### Масштабирование (месяц 2)
1. **Лучший креатив ×2 бюджет**
2. **Добавить ретаргет** на скачавших
3. **Добавить look-alike 2% / 5%**
4. **Бюджет:** 700-1000 ₽/день

### Оптимизация (месяц 3)
1. **Отключить слабые креативы** (CPL > 400)
2. **Лучший креатив ×3 бюджет**
3. **Добавить новые форматы** (видео, карусель)
4. **Тест ботов в рекламе** (для перехода в TG)

## Пиксель VK

### Установка
```html
<!-- Код пикселя на лендинг -->
<script type="text/javascript">
  // ... код пикселя VK ...
</script>
```

### События
- `lead` — скачал магнит
- `subscribe` — оставил email
- `pay` — оплатил
- `view` — посетил страницу
- `add_to_cart` — добавил в корзину

### Цели
- Настроить оптимизацию по конверсиям (CPA)
- Ретаргет на нужные сегменты

## Запуск за 5 шагов

1. **Создать кабинет** (юр. лицо / НПД + ИНН + ERID)
2. **Подключить пиксель** (на лендинг)
3. **Загрузить аудиторию** (подписчики группы, база email)
4. **Создать кампанию** (выбрать цель → сегмент → креатив)
5. **Запустить тест** (3-5 дней, 300-500 ₽/день)

## Сравнение с ОК

|----------|------------|------------|
| CPL (магнит) | 80-250 ₽ | 40-120 ₽ |
| CPL (подписка) | 600-1500 ₽ | 400-1000 ₽ |
| Лучший сегмент | 25-44 | 35-55 |
| Look-alike | ✅ (от 100) | ✅ (от 100) |
| Видеоклипы | ✅ (1 млрд MAU) | ✅ (36 млн MAU) |
| Сторис | ✅ | ✅ |

**Вывод:** ОК дешевле для ж 35-55, VK — для ж 25-44.

## Топ-3 ошибки в VK Реклама

1. **Слишком узкий таргетинг** (< 10 000 чел. — алгоритм не сходится)
2. **Один креатив на месяц** — выгорает за 1-2 недели
3. **Нет пикселя** — нет ретаргета, нет оптимизации
