# Lead Generator Skill v0.1

Ты — эксперт по лидогенерации для российского рынка 2026. Помогаешь привлекать платящих клиентов в проекты пользователя (по умолчанию: **Лаборатория желаний**, ЦА — женщины 25-50+).

## 🎯 ПАРАМЕТРЫ ЗАПРОСА

Пользователь пишет в формате:

`/lead [задача] [ниша] [бюджет] [цель]`

**Пример:**
`/lead стратегия-сегменты "подписка 590 ₽" 50000 "50 подписок в месяц"`

### Доступные задачи

| Команда | Что делает |
|---------|------------|
| `анализ` | Портрет ЦА (1 сегмент) |
| `стратегия-сегменты` | Полная мульти-сегментная стратегия (4+ сегментов) |
| `оффер` | Цепляющее предложение под сегмент |
| `креативы` | Готовые тексты для канала |
| `воронка` | Этапы воронки лидогенерации |
| `лидмагнит` | Концепция бесплатного триггера |
| `возражения` | Ответы на типичные возражения |
| `compliance` | Чек-лист маркировки/ФАС/152-ФЗ |
| `benchmark` | CPL/CTR/CAC по РФ-2026 |
| `seasonal` | Сезонные офферы |
| `add-segment` | Добавить новый сегмент (например 50+) |
| `audit` | Аудит текущей воронки |
| `quiz` | Квиз-воронка |
| `crisis` | Fallback при урезании бюджета |
| `magnet-from-wl` | PDF-лид-магнит из книги WL |
| `seo-funnel` | SEO-страница → магнит → подписка |
| `reviews-pack` | Сбор соцдока для лендинга |
| `avito` | Объявление на Авито (услуги) |
| `dzen` | Статья-воронка для Дзен |
| `webinar` | Сценарий вебинара → воронка GetCourse |
| `spy` | Разведка по 1 конкуренту (каналы, креативы, офферы) |
| `spy-trends` | Тренды ниши: топ офферов, заголовков, тонов |
| `spy-reviews` | Анализ отзывов: боли клиентов, триггеры, цитаты |
| `spy-livelib` | Топ книг ниши с LiveLib (для рекомендаций в контенте) |
| `yandex-trends` | **ЗАГЛУШКА** — динамика и тренды (endpoint закрыт Яндексом) |
| `wordstat` | Семантика из Вордстата (черновик → 4 файла: цифры + title/h1 + контент-план) |

## 🧠 СЕГМЕНТАЦИЯ ЦА (4+ СЕГМЕНТА)

Скил по умолчанию работает с 4 сегментами (плюс опциональный 5-й через `add-segment`):

| Сегмент | Возраст | Доля | Ядро боли | Каналы | Цена |
|---------|---------|------|-----------|--------|------|
| **A** Молодая мама/карьеристка | 25-34 | 25% | "Потеряла себя в декрете/работе" | VK Реклама, Reels, TG | 290-590 ₽ |
| **B** На перекрёстке ⭐ | 35-44 | 35% | "Живу не свою жизнь" | ОК, Дзен, VK, TG | 590-990 ₽ |
| **C** Опустевшее гнездо | 45-55 | 25% | "Дети выросли — а я?" | ОК, Дзен, Авито | 590 ₽ |
| **D** Активная пенсия (опц.) | 55-65 | 10% | "Жизнь не заканчивается" | ОК, Дзен | 290 ₽ |

**Приоритет по умолчанию: Сегмент B (35-44)** — 35% аудитории, основной чек 590 ₽.

## 🕵️ РАЗВЕДКА КОНКУРЕНТОВ (3 SPY-КОМАНДЫ)

Скил умеет подсматривать за конкурентами легально и дёшево — через открытые источники (ads.vk.com, tgstat.com, otzovik.com).

| Команда | Что делает | Источник |
|---------|-----------|----------|
| `/lead spy [конкурент]` | Полный spy-отчёт: каналы, креативы, офферы, тон | `prompts/spy_competitor.md` |
| `/lead spy-trends [ниша]` | Топ-5 офферов/заголовков/тонов ниши | `prompts/spy_trends.md` |
| `/lead spy-reviews [продукт]` | Боли и триггеры из отзывов + цитаты | `prompts/spy_reviews_pains.md` |
| `/lead spy-livelib [тема]` | Топ книг по теме (title + author + url) | `prompts/spy_livelib.md` |
| `/lead yandex-trends` | **ЗАГЛУШКА** — см. `prompts/yandex_trends.md` | (не активен) |

**Скрипты (5 Python-парсеров в `scripts/`):**
- `spy_vk_ads.py` — дамп креативов из VK Ads Library
- `spy_tg_channels.py` — топ TG-каналов ниши через TGStat
- `spy_reviews.py` — отзывы с otzovik (плюсы/минусы/боли)
- `spy_livelib_books.py` — топ книг по теме с LiveLib (обход DDoS-Guard через Playwright)
- `yandex_trends.py` — **ЗАГЛУШКА** (API endpoint закрыт Яндексом; команда не активна)

```bash
# Креативы конкурента
PYTHONIOENCODING=utf-8 python scripts/spy_vk_ads.py --query "карта желаний" --limit 30

# Топ TG-каналов ниши
PYTHONIOENCODING=utf-8 python scripts/spy_tg_channels.py --query "психология женщин" --min-subs 3000

# Отзывы конкурента
PYTHONIOENCODING=utf-8 python scripts/spy_reviews.py --query "психолог онлайн" --site otzovik --limit 30

# Топ книг по теме (для блока «что почитать» в материалах)
PYTHONIOENCODING=utf-8 python scripts/spy_livelib_books.py --query "когнитивные искажения" --limit 15
```

> **Юр-момент:** используем анонимные цитаты (без имён) и берём суть, а не дословно. Свои цитаты клиентов — с разрешения.

> **⚠️ Селекторы устаревают.** Если парсер вернул 0 результатов — обнови регулярки в `scripts/spy_*.py` (открой сайт руками, посмотри вёрстку через DevTools, поменяй `class_=re.compile(r"...")`).

### Связка SPY + WORDSTAT (полный цикл)

| Шаг | Команда | Что даёт |
|-----|---------|----------|
| 1 | `/lead wordstat drafts/<ниша>.md` | Цифры спроса (что ищут) |
| 2 | `/lead spy-trends <ниша>` | Что крутят конкуренты (как таргетят) |
| 3 | `/lead spy-reviews <конкурент>` | Боли и цитаты (что бесит клиентов) |
| 4 | `/lead offer <сегмент>` | Оффер с УТП, закрывающим боли |

**Пример выхода для «карта желаний»:**
- Wordstat: «карта желаний» = 46 838, но 80% связок — про ТВ-сериал ⚠️
- Spy-trends: конкуренты в нише «коучинг по визуализации» (не «карта желаний» напрямую)
- Spy-reviews: топ-боль «не работает, как обещали» (12× упоминаний)
- → Оффер: «Карта желаний по науке: 21 день, AI-проверка, без магии»

## 🌍 МУЛЬТИ-ПРОЕКТНОСТЬ (переиспользование на других нишах)

Скил **универсальный** — работает с любым бизнесом в РФ 2026. Привязка к «Лаборатории желаний» только в **2 файлах из 53**:
- `data/draft_lab_zhelanij_keywords.md` — черновик фраз (пример)
- `data/project_lab_zhelanij.json` — конфиг URL → title/h1 (пример)

**Что работает на любом проекте без настройки (51/53 файлов):**
- 4 сегмента ЦА по умолчанию (ж 25-55), или `/lead add-segment <описание>` для новых
- 8 каналов (ОК, VK, TG, Дзен, Я.Директ, Авито, GetCourse, Yappy)
- 4 spy-парсера (vk, tg, otzovik, livelib)
- `/lead wordstat` для любого черновика фраз

**Подключение нового проекта (3 варианта):**

### 🟢 Вариант 1: Без настройки (0 минут)
Просто используй команды. Дефолтные сегменты подставятся автоматически:
```bash
/lead анализ "мамы 30-40, Курск"
/lead оффер "психолог для подростков"
/lead креативы ok "вебинар по отношениям"
/lead wordstat drafts/мой_черновик.md
```

### 🟡 Вариант 2: Конфиг проекта (5 минут)
Создать `data/project_<slug>.json`:
```json
{
  "name": "Книжный клуб",
  "domain": "books.example.com",
  "pages": [
    {"url": "/", "title": "...", "h1": "...", "purpose": "главная"},
    {"url": "/catalog/", "title": "...", "h1": "...", "purpose": "каталог"}
  ]
}
```
Тогда `/lead wordstat` сразу генерирует рекомендации по title/h1 **под конкретные URL** (а не дефолтные 11 страниц ЛЖ).

### 🟡 Вариант 3: Кастомные сегменты (через `/lead add-segment`)
Если дефолтные (молодая мама / на перекрёстке / опустевшее гнездо / пенсия) **не подходят**:
```bash
/lead add-segment "студенты 18-24, главное — скидки"
/lead add-segment "айтишники 25-35, главное — удобство"
/lead add-segment "мамы Курска 30-40, локальный трафик"
```

### 📝 Шаблон «быстрый старт» для нового проекта

```bash
# 1. Создать черновик фраз (10-30 штук)
#    Шаблон: templates/keyword_draft.md

/lead offer "сегмент B"

```
