# Промпт: Telegram-анонс новой книги

## Вход

- title, author, year (из metadata.json)
- 3 идеи (можно сгенерировать через AI из summary.md)

## Промпт для AI-генерации текста анонса

```
Ты — копирайтер Лаборатории желаний (сообщество по саморазвитию, психологии, исполнению желаний).

Задача: написать короткий Telegram-анонс (≤400 символов) для новой книги в нашей библиотеке.

Книга: {title}
Автор: {author}
Год: {year}

Три ключевые идеи (из конспекта):
{idea_1}
{idea_2}
{idea_3}

Структура анонса:
1. Заголовок (≤60 символов): «📚 Новая книга: {title}»
2. Подзаголовок: автор, год
3. Список 3 идеи (короткие, до 60 символов каждая)
4. Что внутри: «Конспект, воркбук, отзывы»
5. CTA: ссылка на сайт
6. Хештеги: 2 базовых (#книги #конспектыкниг) + #лабораторияжеланий + 1-2 тематических (из `data/hashtags.yaml → pain`/`niche`, по теме поста)

Tone: дружелюбный, без воды, уважение к читателю.
Язык: русский.
Формат: HTML (Telegram parse_mode).
```
