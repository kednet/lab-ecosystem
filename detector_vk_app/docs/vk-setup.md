# Привязка URL к VK Mini App (app_id 54640127)

Пошаговая инструкция для kfigh — после того, как файл мини-аппа задеплоен и отдаёт HTTP 200.

## 1. Убедиться, что мини-апп задеплоен

```bash
curl -I https://app.pulab.ru/detector-app/
```

Ожидаемый ответ:
```
HTTP/2 200
server: nginx
content-type: text/html
```

Если 404 — `dist/detector-app/index.html` ещё не залит (вернись к `README.md`, секция «Деплой»).

## 2. Открыть кабинет приложения

Перейти: **https://vk.com/editapp?id=54640127**

Это настройки именно того приложения Detector, что виден в сообществе «ЛАБОРАТОРИЯ ЖЕЛАНИЙ» (id=237295798).

В левом меню — вкладки: **Информация**, **Размещение**, **Оформление**, **Отображение**, **Правовые документы**, **Другие площадки**, **Дополнительные**, **Разработка**, **Модерация**, **Фичеринг**.

## 3. Вкладка «Размещение»

Перейти на вкладку **«Размещение»** в левом меню.

В форме должно быть одно или несколько полей:
- **«Адрес приложения»** (или «URL», «Ссылка на приложение»)
- (опционально) **«Домен»** — если VK разделяет домен и путь

Прописать:
- **Адрес**: `https://app.pulab.ru/detector-app/`
- **Домен** (если есть): `app.pulab.ru`

Нажать **«Сохранить»** (или «Применить»).

## 4. Дождаться модерации (если требуется)

Если после сохранения появится плашка «На модерации» — VK проверит приложение в течение нескольких часов/дней (обычно 1-3 дня для нового приложения). Пока идёт модерация, мини-апп может не открываться через `https://vk.com/app54640127`, но будет доступен по прямой ссылке в кабинете (есть кнопка «Открыть» или «Предпросмотр»).

## 5. Проверить в мобильном ВК

Открыть в мобильном приложении ВКонтакте ссылку:
```
https://vk.com/app54640127
```

Или через само сообщество «ЛАБОРАТОРИЯ ЖЕЛАНИЙ» (https://vk.com/club237295798) → правый блок «Приложения» → Detector → **«Запустить»**.

Должен открыться наш мини-тест: интро с кнопкой «Начать тест», 5 вопросов, финальный экран с % навязанного и шарингом.

## 6. Что проверить внутри мини-аппа

Внутри ВК-клиента должны работать:
- **VKWebAppInit** (без ошибок в DevTools) — если открыт «Предпросмотр», в консоли видно `vk-bridge: initialized`
- **Тактильная обратная связь** — на телефоне при выборе ответа должна быть лёгкая вибрация
- **Шаринг «ВКонтакте»** — открывает нативный диалог ВК (не web-окно)
- **Скопировать текст** — копирует в буфер обмена

В web-версии (`https://app.pulab.ru/detector-app/` в обычном браузере) всё то же, но без вибрации и шаринг идёт через `vk.com/share.php` (fallback).

## 7. Если что-то не работает

| Симптом | Что делать |
|---|---|
| 404 / белый экран в ВК | Проверить `curl -I https://app.pulab.ru/detector-app/` — должно быть 200. Проверить, что в кабинете VK в «Размещении» указан именно `https://app.pulab.ru/detector-app/` (с трейлинг-слешем и `https://`) |
| «Нарушение политики» / блок | VK модерирует после первой публикации. Подождать 1-3 дня, обычно без последствий для белого приложения |
| `ReferenceError: vkBridge is not defined` | VK Bridge не загрузился. Проверить, доступен ли `https://unpkg.com/@vkontakte/vk-bridge/dist/browser.min.js` (в РФ может быть заблокирован). Внутри ВК скрипт инжектится автоматически. Если ошибка в обычном браузере — это нормально, код обернут в `if (window.vkBridge)` |
| Мини-апп открывается, но кнопки не реагируют | Открыть «Предпросмотр» с `?dev=1` или через DevTools ВК — посмотреть console errors |
| Другое | Написать kfigh → Claude для разбора |

## 8. Когда всё работает

- Проверить, что в `wish_librarian/.env` есть `VK_MINI_APP_ID=54640127` (там уже после правки 2026-06-19)
- Обновить `MEMORY.md` (если ещё не сделано): запись про Detector Mini App
- Можно публиковать в сообщество пост с ссылкой `https://vk.com/app54640127` — кнопка «Запустить» появится в карточке приложения
- Идти итерировать: backend для сохранения результатов, кнопка в меню сообщества, иконка 512×512, etc.
