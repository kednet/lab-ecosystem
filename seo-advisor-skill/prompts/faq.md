# FAQ генератор (People Also Ask + FAQPage Schema)

## Зачем FAQ
Google показывает блок **«Другие также спрашивают»** (PAA) над/под основной выдачей. Если ты отвечаешь на эти вопросы на своей странице + размечаешь через `FAQPage` schema — попадаешь в:
- **Featured snippet** (position 0)
- Расширенный сниппет с раскрывающимися вопросами
- Голосовой поиск

**Эффект:** рост CTR до 30%, рост видимости в zero-click поиске.

## Алгоритм генерации FAQ

### Шаг 1. Определи seed
- Тема страницы (например, «исполнение желаний»)
- Или URL — тогда WebFetch → текущая тема

### Шаг 2. Сгенерируй 5-10 вопросов
Форматы вопросов (чередуй):

| Формат | Пример |
|--------|--------|
| **Как...?** | Как работает исполнение желаний? |
| **Что такое...?** | Что такое трансерфинг реальности? |
| **Можно ли...?** | Можно ли исполнить желание за 1 день? |
| **Сколько...?** | Сколько времени нужно, чтобы желание исполнилось? |
| **Почему...?** | Почему не исполняются желания? |
| **Когда...?** | Когда лучше загадывать желания? |
| **Стоит ли...?** | Стоит ли читать Трансерфинг реальности? |
| **Что делать если...?** | Что делать, если желание не исполняется? |
| **Где...?** | Где купить книгу Трансерфинг? |
| **Кто...?** | Кто такой Вадим Зеланд? |

**Правила:**
- 5-10 вопросов на страницу
- Начинаются с вопросительного слова
- Содержат ключ/LSI из семантического ядра
- Конкретные, не абстрактные
- Соответствуют интенту страницы

### Шаг 3. Дай ответ
К каждому вопросу — **короткий ответ 40-80 слов**:
- Прямо в первой строке (Google любит 40-60 слов)
- Содержит главный ключ
- Без воды, по существу
- Со ссылками на авторитетные источники, если уместно

**Формула ответа:**
> **Прямой ответ (1-2 предложения)**. {Деталь/уточнение}. {Пример или доказательство}. {Связь с темой страницы}.

### Шаг 4. Разметь через Schema FAQPage
См. `prompts/schema.md` → раздел 3. Вставь в `<head>` или перед `</body>`.

### Шаг 5. Сделай видимый FAQ-блок на странице
```html
<section class="faq" itemscope itemtype="https://schema.org/FAQPage">
  <h2>Часто задаваемые вопросы</h2>
  <div itemscope itemprop="mainEntity" itemtype="https://schema.org/Question">
    <h3 itemprop="name">{вопрос}</h3>
    <div itemscope itemprop="acceptedAnswer" itemtype="https://schema.org/Answer">
      <p itemprop="text">{ответ 40-80 слов}</p>
    </div>
  </div>
  ...
</section>
```

## Источники вопросов
1. **Google SERP** — ввести seed, посмотреть блок «Другие также спрашивают» (если есть WebFetch)
2. **Яндекс SERP** — аналогично, плюс блок «Что люди ищут» внизу
3. **AnswerThePublic** (answerthepublic.com)
4. **Related Searches** внизу SERP
5. **Форумы**: vc.ru, reddit, pikabu, woman.ru — какие вопросы задают реальные люди
6. **Комментарии** в твоём VK/Telegram
7. **Поддержка** — какие вопросы приходят от читателей

### Блок 2: расширенный FAQ (3-5 вопросов)
Для вторичной выдачи, более глубокие вопросы.

### Schema FAQPage (готовый JSON-LD)
```json
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [
    {
      "@type": "Question",
      "name": "{вопрос 1}",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "{ответ 1}"
      }
    },
    ...
  ]
}
```

### H2 на странице
```html
<h2>Часто задаваемые вопросы</h2>
```

### Чек-лист внедрения

```

## Готовый пример для темы «Исполнение желаний»

### Вопрос 1: Как работает исполнение желаний?
**Ответ:** Исполнение желаний работает через фокусировку внимания на цели и эмоциональное проживание результата как уже свершившегося факта. Мозг не различает реальный и воображаемый опыт, поэтому регулярная визуализация запускает те же нейронные цепочки, что и реальное действие. Ключевые практики — карта желаний, аффирмации и медитации.

### Вопрос 2: Сколько времени нужно, чтобы желание исполнилось?
**Ответ:** Сроки зависят от сложности желания и регулярности практики. Простые цели (сходить в кино, купить вещь) могут реализоваться за 1-2 недели. Глобальные изменения (отношения, карьера) — от 3 месяцев до 1-2 лет. Главный фактор — внутренняя готовность, а не календарь.

### Вопрос 3: Можно ли исполнить желание силой мысли за 1 день?
**Ответ:** Маловероятно для больших целей. За 1 день можно сдвинуть инертность — принять решение, написать запрос, сделать первый шаг. Но «материализация» за день — это миф. Подлинное исполнение требует перестройки мышления, привычек и образа действий, что занимает время.

### Вопрос 4: Что делать, если желание не исполняется?
**Ответ:** Проанализируйте 4 фактора: 1) действительно ли это ваше желание, а не навязанное; 2) какова эмоция при визуализации — радость или тревога; 3) есть ли внутренние блоки (страх, чувство вины, «не заслуживаю»); 4) какие действия вы предпринимаете. Часто «неисполнение» — это защита от неподходящей цели.

### Вопрос 5: Какие книги про исполнение желаний лучшие?
**Ответ:** Топ-3 для старта: «Трансерфинг реальности» Вадима Зеланда (техника управления реальностью через мысли), «Сила настоящего» Экхарта Толле (проживание текущего момента) и «Подсознание может всё» Джона Кехо (метод визуализации). В нашей библиотеке есть подробные конспекты каждой из этих книг.

## Анти-паттерны FAQ
- ❌ Маркетинговые вопросы («Почему наша компания лучшая?»)
- ❌ Вопросы без ответа или с ответом «узнайте больше»
- ❌ Скрытый текст (Google банит)
- ❌ Дублирование вопросов с разной формулировкой
- ❌ Слишком длинные ответы (>200 слов теряют шанс на snippet)
