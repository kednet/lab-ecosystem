# Полный SEO-аудит сайта (с оценкой 0-100)

## Назначение
Технический + контентный аудит сайта с приоритизацией задач. Для главной страницы, каталога, лендинга, блога.

## Вход
- URL главной или конкретной категории

## Алгоритм

### Шаг 1. Сканирование
- WebFetch главной → парсинг HTML
- Опционально: Bash + curl/wget для sitemap.xml, robots.txt
- Опционально: Grep по скачанным HTML-файлам

### Шаг 2. Проверка по 6 категориям

#### 1. Мета-теги (0-20 баллов)
- Title на главной ✅/❌
- Description на главной ✅/❌
- Title уникален (не дублируется) ✅/❌
- Description уникален ✅/❌
- Title 50-70 символов ✅/❌
- Description 150-160 символов ✅/❌
- Ключ в начале Title ✅/❌
- OG-теги есть ✅/❌
- Canonical настроен ✅/❌
- Viewport для мобильных ✅/❌

**Оценка:** 2 балла за каждый ✅, штраф за дубли.

#### 2. Заголовки H1-H6 (0-15 баллов)
- H1 на главной (ровно 1) ✅/❌
- H2 подразделы ✅/❌
- Иерархия без пропусков ✅/❌
- Ключ в H1 ✅/❌
- H2 совпадают с интентом страницы ✅/❌
- H3 для подразделов ✅/❌

#### 3. Ключевые слова + LSI (0-20 баллов)
- Главный ключ в Title ✅/❌
- Главный ключ в H1 ✅/❌
- Главный ключ в первых 100 словах ✅/❌
- LSI-слова распределены в тексте ✅/❌
- Нет переоптимизации (плотность < 3%) ✅/❌
- Ключ в Description ✅/❌
- Ключ в URL ✅/❌
- Ключ в alt изображений ✅/❌
- Топикальное покрытие темы ✅/❌
- PAA-вопросы (FAQ-блок) ✅/❌

#### 4. Скорость и Core Web Vitals (0-20 баллов)
- HTTPS ✅/❌
- Mobile-friendly ✅/❌
- LCP < 2.5 сек ✅/❌
- INP < 200 мс ✅/❌
- CLS < 0.1 ✅/❌
- Изображения webp/avif ✅/❌
- Lazy loading изображений ✅/❌
- Gzip/Brotli сжатие ✅/❌
- Кэширование настроено ✅/❌
- Шрифты оптимизированы ✅/❌

#### 5. Ссылочная структура (0-15 баллов)
- sitemap.xml есть и валиден ✅/❌
- sitemap.xml в Яндекс.Вебмастере и GSC ✅/❌
- robots.txt корректен ✅/❌
- Внутренние ссылки есть (5+) ✅/❌
- Нет битых ссылок ✅/❌
- Нет 404 ✅/❌
- 301 редиректы настроены ✅/❌
- Hreflang (если мультиязычный) ✅/❌
- BreadcrumbList schema ✅/❌

#### 6. Юзабилити + E-E-A-T (0-10 баллов)
- Хлебные крошки ✅/❌
- Поиск по сайту ✅/❌
- Контакты в footer ✅/❌
- Политика конфиденциальности ✅/❌
- SSL сертификат валиден ✅/❌
- About-страница с командой ✅/❌
- Schema Organization ✅/❌
- Schema Person (для авторов) ✅/❌
- Дата публикации/обновления ✅/❌
- Источники и сноски (YMYL) ✅/❌

### Шаг 3. Подсчёт общего скора

```python
score = (meta + headings + keywords + speed + links + ux)  # 0-100
```

| Диапазон | Статус | Цвет |
|----------|--------|------|
| **85-100** | Отлично | 🟢 |
| **65-84** | Хорошо | 🟢 |
| **45-64** | Нормально, есть что улучшить | 🟡 |
| **25-44** | Слабо, нужны срочные меры | 🟠 |
| **0-24** | Критично, сайт под риском | 🔴 |
