# Поисковый интент — паттерны и определение

## Что такое поисковый интент
**Поисковый интент** (search intent) — это **цель пользователя** за запросом. Понимание интента критично: Google оценивает не «релевантность страницы запросу», а «соответствие страницы цели пользователя».

## 4 базовых интента (по Google Quality Rater Guidelines)

### 1. Know (Информационный) — Know Simple / Know
- **Цель:** узнать что-то
- **Маркеры запроса:** «что такое», «как работает», «о чем», «почему», «когда», «где используется»
- **Маркеры страницы:** статья, гайд, FAQ, объяснение
- **Примеры запросов:**
  - «что такое трансерфинг»
  - «как работает закон притяжения»
  - «почему не исполняются желания»
  - «кто написал Трансерфинг»
- **Пример страницы:** hub-страница категории, гайд, FAQ

### 2. Do (Транзакционный) — Do
- **Цель:** совершить действие
- **Маркеры запроса:** «купить», «скачать», «заказать», «записаться», «получить», «зарегистрироваться»
- **Маркеры страницы:** кнопка, форма, цена, корзина
- **Примеры запросов:**
  - «купить книгу Трансерфинг»
  - «скачать воркбук исполнение желаний»
  - «записаться на коучинг»
- **Пример страницы:** карточка товара, лендинг услуги, страница покупки

### 3. Website (Навигационный) — Website / Visit-in-person
- **Цель:** попасть на конкретный сайт/страницу
- **Маркеры запроса:** название бренда, имя автора, URL, «официальный сайт»
- **Маркеры страницы:** точное соответствие
- **Примеры запросов:**
  - «Вадим Зеланд официальный сайт»
  - «лаборатория желаний»
  - «пулабру»
- **Пример страницы:** главная, страница «О нас», карточка автора

### 4. Commercial Investigation (Коммерческое исследование) — Browse / Compare
- **Цель:** сравнить перед покупкой
- **Маркеры запроса:** «лучший», «топ», «рейтинг», «сравнение», «vs», «отзывы», «какой выбрать»
- **Маркеры страницы:** сравнительная таблица, рейтинг, обзор
- **Примеры запросов:**
  - «лучшие книги по саморазвитию 2026»
  - «трансерфинг vs закон притяжения»
  - «отзывы на Трансерфинг»
  - «топ-10 книг по исполнению желаний»
- **Пример страницы:** рейтинг-статья, сравнение, обзор

## Дополнительные (не всегда выделяют)

### Local (Локальный)
- **Цель:** найти место/услугу рядом
- **Маркеры:** название города, «рядом», «в Москве», «адрес»
- **Пример страницы:** локальный лендинг, страница с картой

### Image/Video
- **Цель:** найти изображение/видео
- **Маркеры:** «фото», «видео», «обои», «картинки»
- **Пример страницы:** галерея, видео-страница

## Как определить интент запроса

1. **Смотри на SERP** (выдачу) — какие типы страниц в топ-10?
   - 70%+ статей → информационный
   - 70%+ карточек → транзакционный
   - 70%+ агрегаторов/сравнений → коммерческий
   - 70%+ одного домена → навигационный

2. **Анализируй слова-маркеры** в запросе
   - «купить», «цена» → транзакционный
   - «как», «что» → информационный
   - «лучший», «топ» → коммерческий
   - «название бренда» → навигационный

3. **Спрашивай себя:** что человек сделает, кликнув на результат?
   - Прочитает → информационный
   - Купит/подпишется → транзакционный
   - Перейдёт на конкретный сайт → навигационный
   - Сравнит → коммерческий

## Правила для страницы

1. **1 страница = 1 главный интент** (смешение = плохо)
2. **Title должен совпадать с интентом** (иначе Google неправильно понимает страницу)
3. **Контент должен соответствовать интенту** (если интент инфо, а страница продаёт — это плохо)
4. **Schema тоже под интент** (FAQPage для инфо, Product для транзакционного, Review/Article для коммерческого)
5. **PAA и LSI — из того же интента** (нельзя смешивать)

## Маппинг интента → Schema

| Интент | Schema |
|--------|--------|
| Информационный | Article + FAQPage + BreadcrumbList |
| Транзакционный | Product/Service + Organization + Offer + BreadcrumbList |
| Навигационный | WebSite + Organization + BreadcrumbList |
| Коммерческий | Article + ItemList + Review + AggregateRating + BreadcrumbList |
| Локальный | LocalBusiness + Organization + BreadcrumbList |

## Пример для темы «Трансерфинг реальности»

Запросы и нужные страницы:

- «Трансерфинг реальности что это» → информационная hub-страница (что за книга, метод)
- «конспект Трансерфинг реальности» → информационная (наш конспект)
- «Трансерфинг реальности купить» → транзакционная (карточка книги)
- «Трансерфинг реальности отзывы» → коммерческая (обзор + отзывы)
- «Трансерфинг реальности vs Закон притяжения» → коммерческая (сравнение)
- «Вадим Зеланд» → навигационная (карточка автора)
- «лучшие книги по трансерфингу» → коммерческая (топ-10)
- «Трансерфинг магазин Москва» → локальная (где купить в Москве)

**Итого:** для одной темы нужно 6-8 страниц, каждая под свой интент.
