# Стоп-слова для URL-slug (ru + en)

При генерации slug стоп-слова удаляются, чтобы:
- Сократить длину URL
- Убрать шум (предлоги, артикли не несут смысла)
- Сделать slug более «ключевым»

## Русские стоп-слова
```
и, в, на, по, с, со, о, об, от, до, из, за,
для, как, что, это, эта, этот, эти, тот, та, те,
или, но, а, же, бы, ли, ни, так, его, её, их,
к, у, над, под, при, без, через, между, перед,
все, всё, всех, наш, ваш, свой, мой, твой, ещё,
уже, только, даже, потом, теперь, когда, если, чтобы,
кто, где, куда, откуда, зачем, почему, сколько, какой,
имеет, имеет ли, можно ли, нужно ли, стоит ли,
```

## Английские стоп-слова
```
the, a, an, and, or, but, in, on, at, to, for,
of, with, by, from, as, is, was, are, be, been,
this, that, these, those, it, its,
```

## Правила
- Удалять **только** если после удаления slug не становится бессмысленным
- Если ключ = «как бросить курить», не удалять «как» (хотя стоп-слово) — это часть запроса
- Не удалять «не», «без», «против» (могут нести смысл)
- Не удалять слова короче 3 букв (могут быть ключами: «код», «я», «ru»)
- Всегда оставлять минимум 1-2 слова в slug

## Когда НЕ удалять стоп-слова
- «Как перестать» — оставить «как» (без него теряется смысл)
- «Не могу» — оставить «не» (отрицание меняет интент)
- «С кем» — оставить «с» (если это смыслообразующее)

## Алгоритм
1. Транслитерировать
2. Разбить на слова
3. Удалить стоп-слова
4. Если осталось < 2 слов — вернуть 2 главных слова + стоп-слова
5. Соединить через дефис
6. Обрезать до 60 символов по границе слова
