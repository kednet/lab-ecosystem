# Факторы ранжирования Яндекс 2026

## Технические
- **Скорость загрузки** (влияет на поведенческие): < 1.5 сек desktop, < 2.5 сек mobile
- **Mobile-friendly** (обязательно, иначе пессимизация в мобильной выдаче)
- **HTTPS** (с 2019 обязателен для всего сайта)
- **XML sitemap** (с приоритетами и lastmod)
- **robots.txt** (корректный, без блокировки важных страниц)
- **Canonical** (устранение дублей)
- **404** (минимум, корректная страница 404)
- **Редиректы** (301, без цепочек)
- **Hreflang** (для мультиязычных сайтов)
- **Структурированные данные** (Schema.org — поддерживается)

## Контентные
- **Уникальность** (главный фактор; проверка через text.ru, content-watch)
- **Объём** (для информационных — 2000+ знаков; для лендингов — 1000+)
- **Заголовки H1-H6** (иерархия без пропусков, ключ в H1)
- **Title и Description** (оптимизированные, уникальные)
- **Релевантность интенту** (страница должна отвечать на запрос, не просто содержать слова)
- **Свежесть** (для новостных тем — критично, для вечных — вторично)
- **Опыт и экспертиза автора** (E-E-A-T аналог — «доверие к источнику»)
- **Грамотность** (орфография, пунктуация)
- **Полнота покрытия темы** (LSI, топикальная авторитетность)
- **FAQ-блок** (буст в выдаче — Яндекс показывает «Быстрые ответы»)

## Поведенческие (у Яндекса ВЕС БОЛЬШЕ чем у Google)
- **CTR в выдаче** (чем выше — тем лучше для ранжирования)
- **Время на сайте** (длинное = хорошо, но не бесконечное)
- **Глубина просмотра** (сколько страниц за сессию)
- **Показатель отказов** (bounce rate < 30% — хорошо)
- **Возвраты к поиску** (если пользователь вернулся в поиск — плохой сигнал)
- **Прямые заходы** (если люди возвращаются на сайт = хороший сигнал)
- **Брендовый трафик** (запросы «лаборатория желаний» = доверие)

## Ссылочные
- **Внешние ссылки** (с авторитетных тематических сайтов)
- **Анкор-лист** (разнообразие, без переспама «точным вхождением»)
- **Внутренняя перелинковка** (hub-spoke, тематические кластеры)
- **Без битых ссылок**
- **Без покупных ссылок** (Минусинск — фильтр за покупку)

## Гео
- **Региональность** (привязка к региону в Яндекс.Вебмастере)
- **Контакты с региональным адресом** (footer)
- **Schema LocalBusiness** (если локальный бизнес)

## Уникальные для Яндекса
- **Оригинальные тексты** (загрузка в Яндекс.Вебмастер → защита от копирования)
- **Турбо-страницы** (deprecated, но Умный сниппет/AI-обзор — преемник)
- **Быстрые ссылки** (формируются автоматически, можно влиять структурой)
- **Кликабельные сниппеты** (с автоопределением)
- **Подсказки в выдаче** (история запросов пользователя)

## Анти-факторы (штрафы)
- **Минусинск** (покупные ссылки)
- **АГС** (низкокачественный сайт)
- **Переоптимизация** (переспам ключами)
- **Клоакинг** (разный контент для ботов и людей)
- **Скрытый текст** (display:none для ключей)
- **Дорвеи** (страницы-однодневки)
- **Копипаст** (неуникальный контент)
