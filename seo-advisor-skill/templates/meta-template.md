# Шаблоны мета-тегов по типу страницы

## Главная страница
```html
<title>Лаборатория желаний — конспекты книг по саморазвитию и исполнению желаний</title>
<meta name="description" content="Конспекты, воркбуки и рецензии лучших книг по саморазвитию, психологии и исполнению желаний. Читайте кратко, применяйте на практике. Бесплатный доступ.">
<meta name="keywords" content="конспекты книг, саморазвитие, исполнение желаний, воркбук, психология, трансерфинг реальности, личностный рост">
<link rel="canonical" href="https://lab.com/">
```

## Страница книги (конспект)
```html
<title>{Название} — {Автор}: конспект, воркбук и отзывы | Лаборатория желаний</title>
<meta name="description" content="Читайте краткий конспект книги «{Название}» {Автор}. Ключевые идеи, цитаты, выводы и практический воркбук для саморазвития. Подходит тем, кто изучает {тема}.">
<meta name="keywords" content="конспект {название}, {автор} книга, {название} краткое содержание, воркбук {название}, {жанр}, саморазвитие">
<link rel="canonical" href="https://lab.com/library/{slug}/">
```

```html
<title>Трансерфинг реальности — Вадим Зеланд: конспект, воркбук и отзывы | Лаборатория желаний</title>
<meta name="description" content="Читайте краткий конспект книги «Трансерфинг реальности» Вадима Зеланда. Ключевые идеи, цитаты, выводы и практический воркбук для исполнения желаний. Подходит тем, кто изучает трансерфинг.">
<meta name="keywords" content="конспект трансерфинг реальности, вадим зеланд трансерфинг, трансерфинг краткое содержание, воркбук трансерфинг, исполнение желаний книга">

```

## Статья блога
```html
<title>{Тема статьи}: {подзаголовок с выгодой} | Лаборатория желаний</title>
<meta name="description" content="{1-2 предложения о чём статья}. {выгода для читателя}. Практические советы, примеры, пошаговый план.">
<meta name="keywords" content="{тема1}, {тема2}, {тема3}, саморазвитие, лаборатория желаний">
<link rel="canonical" href="https://lab.com/blog/{slug}/">
```

**Пример:**
```html
<title>5 техник исполнения желаний по Трансерфингу: практическое руководство | Лаборатория желаний</title>
```

## Страница эксперта / тренера
```html
<title>{Имя} — {специализация}, {формат работы} | Лаборатория желаний</title>
<meta name="description" content="{Имя} — {регалии}. {Что делает}. Запишитесь на {услугу}: 30 минут, бесплатно. Отзывы, статьи, кейсы клиентов.">
<meta name="keywords" content="{имя} эксперт, {специализация}, {формат}, отзывы, консультация, записаться">

```


## Лендинг услуги
```html
<title>{Услуга}: {выгода} за {срок} | {Имя тренера} | Лаборатория желаний</title>
<meta name="description" content="{Что получит клиент}. {Социальное доказательство: N довольных клиентов}. Запишитесь: 30 минут, бесплатное знакомство, без обязательств.">

<link rel="canonical" href="https://lab.com/services/{slug}/">
```

## FAQ-страница
```html
<title>Часто задаваемые вопросы — {тема} | Лаборатория желаний</title>
<meta name="description" content="Ответы на популярные вопросы о {тема}. {выгода}. Узнайте: {вопрос 1}, {вопрос 2}, {вопрос 3}.">

<link rel="canonical" href="https://lab.com/faq/{slug}/">
```

## Правила генерации title

| Правило | Обоснование |
|---------|-------------|
| Длина 50-70 символов | Google показывает ~60 в десктопе, ~50 в мобильном |
| Ключ в начале | Google придаёт больший вес первым словам |
| Бренд в конце через `\|` | Узнаваемость, не съедает длину |
| Уникальный для каждой страницы | Иначе Google склеит в кластер |
| Без `!!!`, `100%`, `лучший` | Переоптимизация → пессимизация |
| Кириллица | Для РФ-аудитории Яндекс+Google |
| Без CAPS | Выглядит как спам |

## Правила генерации description

| Правило | Обоснование |
|---------|-------------|
| Длина 150-160 символов | Google показывает ~155 |
| Начинай с глагола | «Читайте», «Узнайте», «Скачайте», «Изучите» |
| Включай главный ключ | Подсвечивается жирным в выдаче |
| Призыв к действию | Повышает CTR |
| Без дублирования title | Они могут совпадать, но description — самостоятельный |
| Уникальный | Нельзя копипастить между страницами |

## Правила для keywords

| Правило | Обоснование |
|---------|-------------|
| 5-10 ключей | Google не использует meta keywords для ранжирования, но другие системы — да |
| Без дублей | Уникальные слова |
| Включай бренд | «лаборатория желаний» |
| Длинный хвост | «воркбук трансерфинг реальности» лучше, чем «трансерфинг» |
| Без стоп-слов | Не «и», «в», «на», «по» |
