# Промпт LLM для `setting_line` (intro-фраза)

## Назначение
Генерит 1 предложение (8-15 слов), которое встаёт в intro аудио-трека
между «Здравствуй. Это «Лаборатория желаний».» и телом скрипта.

**Зачем:** убрать штамп «Сегодня мы хотим предложить тебе новый
интересный опыт» (одинаковый для всех 10 треков) и дать каждой книге
свою 1-фразовую «затравку».

## Алгоритм вызова
- Файл грузится функцией `_yandexgpt_setting_line()` в `scripts/llm_adapt.py`
- Frontmatter (между `---` маркерами) парсится для параметров:
  - `temperature` (float) — вариативность ответа. 0.0 = детерминированно,
    1.0 = максимально разнообразно. 0.7 — баланс «живой текст, но не мусор».
  - `max_tokens` (int) — лимит длины ответа LLM. 100 хватает на 1 фразу.
  - `model` — НЕ задаётся здесь, берётся из `YANDEX_GPT_MODEL` в `.env`
    (по умолчанию `yandexgpt-lite`).
- System prompt берётся из блока между `# SYSTEM PROMPT` и `# END SYSTEM PROMPT`.
- Если frontmatter битый / маркеров нет — `RuntimeError`, адаптер падает
  на дефолт «Сегодня у нас новая практика — тихо послушай себя.»

## Как редактировать

### Поменять тон/формат/запреты (system prompt)
1. Меняй текст **между** `# SYSTEM PROMPT` и `# END SYSTEM PROMPT` (маркеры не трогай).
2. Сохрани. Изменения подхватятся при следующем запуске.

### Покрутить вариативность/длину
1. Меняй `temperature` / `max_tokens` в frontmatter.
2. Сохрани.
3. Подсказки:
   - `temperature: 0.3-0.5` — скучнее, но стабильнее (для брендовых треков).
   - `temperature: 0.7-0.9` — живее, иногда «выстреливает» красиво (по умолчанию).
   - `temperature: 1.0` — максимум для YandexGPT, **выше API отвергает (HTTP 400)**.
   - `max_tokens: 60-80` — если хочешь гарантированно 1 фразу (без продолжений).
   - `max_tokens: 150-200` — если готов принять 2-3 варианта (потом выберешь).

> ⚠️ Если YandexGPT вернул HTTP 400 — это `temperature > 1.0`. Снижай до 1.0.

### Проверить
```bash
python scripts/llm_adapt.py data/library/<slug>.yaml \
    --template=intro_outro \
    --out=tmp/<slug>.final.yaml
```
В логе увидишь `[+] setting_line: «...»` — это результат. Если мусор —
либо крути `temperature` вниз, либо править system prompt.

# SYSTEM PROMPT
Ты — копирайтер Лаборатории желаний. Пишешь интро для аудио-трека.
Верни РОВНО ОДНО предложение на русском, 8-15 слов.
Тон: тёплый, без пафоса и без эзотерики. Обращайся на «ты» (как «ты»-форма, не формальный).
Формат: «Сегодня у нас {название книги или суть практики} — {1-2 слова о сути}»
НЕ используй слова: «погрузиться», «трансформация», «путешествие», «магия».
НЕ ставь «?» в конце. Только утверждение.
Верни ТОЛЬКО предложение, без кавычек, без пояснений.
# END SYSTEM PROMPT
