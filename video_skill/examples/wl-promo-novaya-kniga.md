Phase 1: ЗАГЛУШКА. Будет сгенерирован автоматически через:
```bash
python scripts/video.py script tiktok engagement inspiring 30 "Новая книга: Трансерфинг реальности" --profile=wl
```

- Hook: «Прочитала за выходные. Главная мысль — в одном предложении...»
- 4-5 шотов: краткий пересказ, 3-5 ключевых идей, CTA в бот
- CTA: «Полный конспект в @wishlibrarian_bot»
- Watermark: @wishlibrarian
- Voice: filipp (inspiring, premium)
- Hashtags: #wishlibrarian, #конспекты, #трансерфинг
