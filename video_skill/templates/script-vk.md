# Шаблон сценария: VK Клипы

## Использование
Шаблон-каркас для `python scripts/video.py script vk ...`. Генерирует сценарий под VK Клипы (9:16, max 60 сек, тёплая подача).

## Метаданные платформы
- **aspect:** 9:16 (1080×1920)
- **max_duration:** 60 сек
- **safe_zones:** top=200, bottom=350, left=50, right=50
- **default_voice:** alena (female, тёплый)
- **default_music_mood:** tide (спокойный, тёплый)

## Специфика VK Клипов
- Аудитория ВК более зрелая (35-45), тёплая подача
- Тон — middle между TikTok (energy) и YouTube (education)
- Тексты — менее панчи, более разговорные
- Хештеги: #вкклипы + нишевые
- BGM: tide (волны/океан), эмбиент

## Структура каркаса
См. `templates/script-tiktok.md` (идентична), отличается только метаданными платформы.
