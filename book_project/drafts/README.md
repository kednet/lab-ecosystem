# Черновики

Сюда пишутся главы. Принцип zero draft: **пишем не редактируя**.

## Именование
`ch-01-короткое-название.md`, `ch-02-...` и т.д.
Название — рабочее, может меняться.

## Правило одного файла
Когда сел писать — открой только тот файл, над которым сегодня работаешь. Не листай другие. Не сравнивай.

## Не редактировать на ходу
- Опечатки — оставлять.
- Слабые фразы — оставлять, рядом писать `[?]` и идти дальше.
- Дыры в сюжете — писать `[здесь будет сцена про X]` и идти дальше.

Возврат назад в zero draft = потеря импульса. Импульс важнее качества.
