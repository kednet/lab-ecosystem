# Заметки

Сырьё. Сюда — всё, что попадается и может пригодиться:
- цитаты из других книг
- подслушанные диалоги
- образы из снов
- факты, которые могут лечь в текст
- неудачные фразы из черновика (вдруг встанут в другой главе)
- фотографии, ссылки на музыку

## Структура
Свободная. Варианты:
- один большой `inbox.md` — всё подряд, разбираем раз в месяц
- по темам: `obrazy.md`, `dialogi.md`, `fakty.md`, `muzyka.md`
- по героям: `nastya.md`, `otec.md` и т.д.

Начни с `inbox.md`, потом увидишь, как удобно.
