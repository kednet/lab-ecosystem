# Лендинг «ЛАБОРАТОРИЯ ЖЕЛАНИЙ» v2 — «Розовый закат»

## Что внутри
- `index.html` — 5 секций (Hero, Польза, Доверие, CTA, Контакты), Hero с обложкой VK
- `styles.css` — палитра «Розовый закат», адаптив mobile-first
- `app.js` — счётчики, reveal-анимация, мягкий параллакс обложки
- `assets/avatar.jpg` (11 КБ) — аватар сообщества, скачан через VK API
- `assets/cover.jpg` (562 КБ) — обложка сообщества 1590×400 px

## Обложка VK — копирайт
Изображение `assets/cover.jpg` взято из официальной обложки VK-сообщества
`https://vk.com/club237295798` (ЛАБОРАТОРИЯ ЖЕЛАНИЙ / pulabru) через VK API
методом `groups.getById` с токеном VK-бота `wish_librarian` — **использовано
с разрешения владельца сообщества**. Копирайт указан в подвале лендинга.

При перепубликации лендинга в открытый доступ рекомендуется:
1. Заменить обложку на локально-подготовленную (оригинал из сообщества)
2. Или оставить со ссылкой на источник в footer
