# /curate-all
Генерирует черновики для всех 8 сфер (120–160 желаний).

**Использование:** `/curate-all`

**Что делает:**
1. `python scripts/curate_wishes.py --all`
2. Проходит по всем 8 `data/spheres/*.yaml`
3. Сохраняет 8 markdown-файлов в `data/library/_draft-*.md`
4. Показывает сводку: количество желаний по сферам

**Время выполнения:** ~5–10 минут (8 запросов к YandexGPT-lite)

**Следующие шаги:**
- Пользователь ревьюит каждую сферу отдельно (или пачкой)
- `/merge` — слить всё в `wishes_final.json`
