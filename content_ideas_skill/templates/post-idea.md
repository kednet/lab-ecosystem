# Шаблон карточки идеи для VK-поста

```yaml
# ============================================================================
#  CARROT / КАРТОЧКА ИДЕИ
#  content-ideas-skill
# ============================================================================

id: idea-2026-07-01-001            # уникальный ID
created: 2026-07-01T12:34:56       # дата создания
version: 1.0                        # версия формата

# === МЕТА ===
target: vk                          # vk | blog | telegram
rubric: разбор-цитаты               # см. profiles/rubrics.md
priority: high                      # high | medium | low

# === СОДЕРЖАНИЕ ===
title: 5 фраз, которые выдают чужое желание
hook: "«Я просто хочу как все» — и почему это красный флаг"
key_idea: |
  Разобрать 5 типичных формулировок, которые выдают
  навязанное желание под видом своего.
  Показать, как отличить «хочу» от «надо» через язык.
structure_hint: storytelling        # AIDA | PAS | storytelling | list | howto

# === ИСТОЧНИК ===
source:
  type: wl                          # wl | coach | competitor-pain | seasonal | trend
  ref: wl-book-id-123               # конкретная ссылка
  quote: "Если все хотят — почему ты чувствуешь пустоту?"
  reason: "Тема в топе у конкурента X, но под нашим углом"

# === АУДИТОРИЯ ===
audience: ca-zhelanii               # см. profiles/lab-zhelanii-ca.md
tone: B3                            # B-исследующий, интенсивность 3

# === CTA И МЕТРИКИ ===
cta: "А у вас было такое? Расскажите в комментах"
target_metric: комменты             # комменты | репосты | сохранения | переходы

# === ОБОСНОВАНИЕ ===
reasoning: |
  Тема в топе у конкурента X (×3 ER).
  Адаптирована под тон ЛЖ: не «просто делай», а «вот детектор».
  Боли ЦА «не понимаю, чего хочу» прямо в фокусе.

# === ОГРАНИЧЕНИЯ ===
notes: |
  - Не использовать «успешный успех»
  - Избегать нравоучения
  - Цитату вставить в начало, не в конец
  - Без ссылок (только в комментах)
  - Эмодзи: 1–2 штуки максимум

# === ДЕДУП ===
fingerprint: abc123def456           # автогенерация (theme + rubric + hook_kw)
```

## Как использовать

1. Сгенерировано в `generate_ideas.py` → попадает в `data/ideas-bank.json`
2. Конкретную карточку выбираем → отдаём в `export_publisher.py`
3. Publisher рендерит готовый пост по `key_idea` + `hook` + `cta`
4. После публикации: добавляется в `data/competitors/pulabru/` для самоанализа

## Где это хранится

- **Банк:** `data/ideas-bank.json` (все идеи)
- **История:** `data/history.json` (для дедупа, с fingerprint)
- **Выгрузка:** `data/generated/<date>-<pack>.md` (markdown-список карточек)
- **Экспорт в Publisher:** `data/generated/publisher-cards/<id>.yaml`
