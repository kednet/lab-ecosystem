# Правила переформатирования (реформат)

> Как одну идею превратить в пост VK / блог / Telegram с сохранением сути, но под формат площадки.

## Принцип

**Идея = смысловое ядро.** Формат = обёртка. Одна идея → 3 формата (опц.).

## VK ↔ Блог ↔ Telegram

### Длина

| Формат | Мин | Оптимум | Макс |
|---|---|---|---|
| Telegram | 300 | 400–800 | 1500 |
| VK | 600 | 800–1500 | 2000 |
| Блог | 1500 | 2500–4000 | 6000 |

### Структура

#### Telegram
```
[Хук — 1 предложение]

[1 основной абзац, 2-4 предложения]

[CTA — 1 фраза]
```

#### VK
```
[Хук — 1-2 предложения]

[Абзац 1: основная мысль]
[Абзац 2: пример/история]
[Абзац 3: разворот/детектор]

[CTA — 1-2 фразы]
```

#### Блог
```
# H1

Лид (1-2 предложения)

## H2 — раздел 1
Текст + пример

## H2 — раздел 2
Список или подразделы

## H2 — Практика
Пошагово

## H2 — FAQ (3-5 вопросов)

## Заключение + CTA
```

## Правила переформатирования

### Из VK в Блог

**Что добавить:**
1. **H1 и H2 структуру** — разбить на разделы
2. **Раскрыть каждый абзац** — добавить примеры, цитаты WL
3. **Добавить практику** — отдельный раздел «Что с этим делать»
4. **FAQ** — 3-5 вопросов (→ SEO Advisor)
5. **Внутренние ссылки** — на WL, Coach, другие статьи
6. **SEO-мета** — title, description, slug
7. **Визуал** — обложка + 1-2 иллюстрации

**Что убрать:**
- Агрессивный CTA «напишите в комментах»
- Эмодзи (1-2 в блоге, не больше)
- «Ссылка в комментах» — заменить нормальной ссылкой

### Из Блога в VK

**Что оставить:**
- Главную мысль
- 1 сильный пример
- 1 крючок

**Что убрать:**
- Структуру H1/H2 (VK плохо рендерит)
- Длинные списки
- Внутренние ссылки
- SEO-блок

**Что добавить:**
- «Продолжение в блоге — ссылка в комментах»
- Эмодзи-маркеры (1-2)
- Короткие абзацы (1-3 предложения)

### Из VK в Telegram

**Что оставить:**
- Крючок
- 1 основную мысль
- CTA

**Что убрать:**
- Всё лишнее (Telegram = 1 мысль, не 3)
- Эмодзи (если их было > 2, оставить 0-1)
- Длинные примеры

**Что добавить:**
- Markdown-разметку (жирный, курсив) — TG рендерит
- Можно использовать эмодзи как маркеры списков

### Из Блога в Telegram

**Безжалостно сократить:**
- 1 крючок
- 1 мысль (не 5)
- 1 пример
- 1 CTA

**Убрать всё остальное.** Если не влезает в 800 знаков — делить на 2-3 поста.

## Авто-реформат (скрипт v0.2)

`scripts/reformat.py` (планируется):
- вход: `data/ideas-bank.json` (одна идея)
- выход: 3 файла — `vk.md`, `blog-outline.md`, `telegram.md`
- правила — захардкожены (по этому документу)

## Чек-лист реформата

Перед публикацией в VK:
- [ ] Первый абзац цепляет (тест «свёрнутого» поста)
- [ ] Абзацы по 1-3 предложения
- [ ] Без markdown-разметки (или минимум)
- [ ] 1-2 эмодзи максимум
- [ ] CTA — одна мысль
- [ ] Без ссылок в начале

Перед публикацией в блоге:
- [ ] H1 один, с ключом
- [ ] H2 — 2-5 штук
- [ ] Meta description заполнена
- [ ] Alt-тексты у картинок
- [ ] 2+ внутренние ссылки
- [ ] FAQ-блок (если релевантно)
- [ ] CTA в конце

Перед публикацией в Telegram:
- [ ] Укладывается в 800 знаков (или 1500 максимум)
- [ ] 1 мысль, не 3
- [ ] Markdown работает
- [ ] 0-1 эмодзи
- [ ] Без ссылок в тексте (если нужны — внизу)
