# Расширенный портрет ЦА (v0.1)

> Фактический портрет на основе данных. Заполняется после `mine_audience_pains.py`. v0.1 — заглушка.

## Стартовые гипотезы (экспертные, из `lab-zhelanii-ca.md`)

- Женщины 25–45
- Боли: «не понимаю, чего хочу», «выбираю правильное, а не своё»
- Язык: рефлексивный, спокойный, без «успешного успеха»

## Что проверим через данные (после парсинга)

### 1. Демография
- [ ] Реальный возраст (по комментам, если видно)
- [ ] Реальная гео
- [ ] Социальный статус (мамы, карьеристки, в поиске — по косвенным признакам)

### 2. Язык
- [ ] Какие слова использует для «желания» (хочу, мечта, тянет, зовёт, тянет к …)
- [ ] Какие слова для «навязанного» (надо, должна, по правилам, не моё)
- [ ] Какие слова-маркеры узнавания («я тоже», «у меня так же», «всё про меня»)

### 3. Боли
- [ ] Топ-10 болей с цитатами
- [ ] Топ-10 вопросов
- [ ] Топ-10 возражений

### 4. Триггеры
- [ ] Какие темы вызывают «вау, это про меня»
- [ ] Какие темы вызывают отторжение

### 5. Сценарии
- [ ] Типичные жизненные ситуации ЦА (когда она «не понимает, чего хочет»)
- [ ] Что она уже пробовала (книги, терапия, практики)
- [ ] Чего боится (что не даёт начать)

### 6. Контент-привычки
- [ ] Какие паблики VK читает
- [ ] Какие книги (жанры, авторы)
- [ ] Какие форматы любит (длинные тексты, карусели, видео)
- [ ] Когда читает (время суток, дни недели)

## Метод

1. **LLM-классификация** комментов на: вопрос / боль / возражение / инсайт / история
2. **Частотный анализ** формулировок (n-gram)
3. **Тематическое моделирование** (LLM или LDA) — какие темы в комментах
4. **Ручная верификация** — LLM может ошибаться, проверяем топ-20
