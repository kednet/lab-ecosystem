# SKILL: Анализ ЦА (audience mining)

## Зачем

В `profiles/lab-zhelanii-ca.md` — **экспертный** портрет ЦА. Здесь — **фактический**, построенный на данных:
- Реальные слова, которыми ЦА описывает свои боли
- Конкретные вопросы, которые задаёт
- Формулировки, которые цепляют
- Сценарии, в которых она узнаёт себя

**Зачем это:** когда пишешь **её словами**, она узнаёт себя → вовлечение выше.

## Что собираем

### 1. Из комментариев к постам конкурентов
**Самый ценный источник.** ЦА пишет сама, без редактуры.

**Что тянем:**
- Топ-100 комментариев к топ-20 постам конкурентов
- Классификация: вопрос / боль / возражение / инсайт / история / оффтоп
- Частотность формулировок
- «Магниты» — фразы, которые повторяются дословно

### 2. Из нашего паблика
**То же самое, но для своих.** Свои комменты — самый ценный источник, потому что это наша ЦА.

**Что тянем:**
- Комменты к нашим постам
- Классификация
- «Магниты»

### 3. Из тем/вопросов в нашем канале/чате (если есть)
Если есть Telegram-чат, Discord, комменты в блоге — там ЦА пишет развёрнуто.

### 4. Из отзывов на книги WL
Если есть отзывы — там ЦА описывает свой опыт с книгой.

### 5. Из опросов (если проводили)
Если был опрос в VK или на сайте — там прямые ответы.

## Как используем

### Pain-language-bank.md
Главный выход. Содержит:
- Топ-10 болей (с цитатами)
- Топ-10 вопросов (формулировками)
- Топ-10 возражений
- «Магниты» — фразы, которые повторяются

→ Используется в `generate_ideas.py` как prompt context.

### Hooks-from-comments.md
20-30 формулировок, которые можно использовать в крючках.

→ Используется в `generate_ideas.py --source pains`.

### CA-deep-profile.md
Углублённый портрет на основе данных:
- Как ЦА формулирует свои желания
- Какие слова использует для «своего» vs «чужого»
- Чего боится больше всего
- Что её вдохновляет

→ Обновляет `profiles/lab-zhelanii-ca.md`.

## Юридические нюансы

- ✅ Берём **открытые** комменты (любой пользователь VK видит)
- ✅ Используем **обезличенно** (не указываем авторов)
- ⚠️ Цитаты с указанием автора — только с его согласия
- ❌ Не собираем **личные данные** (имена, фото, контакты)

## Метрика качества insights

| Метрика | Что показывает |
|---|---|
| Покрытие ЦА | сколько % наших подписчиков найдут «себя» в pain-bank |
| Новизна | сколько новых болей мы нашли (не дублируют expert-гипотезы) |
| Использование | % сгенерированных идей, которые ссылаются на pain-bank |

## Частота обновления

- **Экспертный профиль (lab-zhelanii-ca.md):** раз в квартал
- **Фактический (ca-deep-profile.md):** раз в месяц
- **Pain-bank:** раз в 2–4 недели
- **Hooks:** постоянно (дополняется)
