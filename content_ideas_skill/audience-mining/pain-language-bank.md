## Как использовать

1. **При генерации идей:** `generate_ideas.py --source pains --rubric "детектор"`
2. **В prompt для LLM:** добавить топ-5 болей как контекст
3. **При написании поста:** использовать цитаты ЦА в крючках
4. **В контент-плане:** следить, чтобы каждая рубрика закрывала хотя бы одну боль
