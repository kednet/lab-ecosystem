# Tracker: список конкурентов для мониторинга

> v0.1 — заглушка. Нужно согласовать список 5–10 конкурентов.

## Типы конкурентов

### A. Прямые (психология желаний / самоопределение)
- **Кто ищет ту же ЦА с похожим посылом**
- Мониторим **каждый пост**
- Используем для: рубрикатор, формулировки, анти-рубрикатор

### B. Косвенные (психология, self-help, отношения)
- **Кто ищет ту же ЦА, но с другим фокусом**
- Мониторим **топ-20 постов в неделю**
- Используем для: тренды, форматы, язык ЦА

### C. Книжные паблики (схожие по духу WL)
- **Кто разбирает книги**
- Мониторим **топ-10 постов в неделю**
- Используем для: идеи из книг, цитаты, рубрика «разбор»

## Что выяснить перед добавлением

1. **Реальный URL сообщества** (vk.com/...)
2. **Сколько подписчиков** (не менее 1000, иначе нерепрезентативно)
3. **Как часто постят** (минимум 2 раза в неделю)
4. **ER в норме** (минимум 2% в среднем)
5. **Не «мёртвый» паблик** (активность в комментах)

## Источники для поиска конкурентов

1. **Поиск VK** по ключевым словам:
   - «психология желаний»
   - «женская психология»
   - «отношения и самоопределение»
   - «психология выбора»
   - «женское предназначение» (осторожно — не наш тон)
   - «дневник желаний»
   - «внутренний ребёнок»
   - «психология тела»

2. **Поиск через Яндекс**:
   - «психологические сообщества VK»
   - «паблики для женщин 30+»

3. **Из рекламы VK** — кто рекламируется в нашей нише

4. **Из подписок нашей ЦА** — посмотреть, на кого подписаны наши читатели

5. **Из топов LiveLib / Литрес** — кто ведёт активность по книгам WL
