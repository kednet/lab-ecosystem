# Шаблон карточки эксперта (Markdown)

## Назначение
Используется в подскиле `experts.md` режим `card` для генерации человекопонятной карточки эксперта.

## Шаблон

```markdown
---
slug: {expert-slug}
type: expert
generated_at: {ISO date}
status: draft | published
---

# {Имя Фамилия}

**{Должность / главная роль}**

> {Краткая цитата или 1-2 предложения о подходе эксперта}

![{Имя}]({avatar_url})

## 📋 Основное

| Поле | Значение |
|------|----------|
| **ФИО** | {Полное имя} |
| **Сфера** | {психология / бизнес / литература} |
| **Специализация** | {что именно делает} |
| **География** | {Москва / РФ / международный} |
| **Языки** | {русский, английский} |
| **Сайт** | [{url}]({url}) |
| **Email** | {только если публичный} |

## 🎓 Образование и регалии

- **{Степень}**, **{ВУЗ}** ({год})
- **{Сертификат}**, **{Организация}** ({год})
- {другая регалия}

## 📚 Публикации

### Книги
- **{Название}** ({год}, {издательство}) — {краткое описание}
- **{Название 2}** ({год}) — {краткое описание}

### Статьи
- «{Название статьи}» — {издание}, {дата} → [{url}]({url})
- «{...}» — ...

## 🎙️ Медиа

### YouTube
- [{Название канала}]({url}) — {X} подписчиков
- Ключевые видео:
  - [{Название видео}]({url}) — {просмотров}
  - [...]

### Telegram
- [{@channel}]({url}) — {X} подписчиков

### VK
- [{Группа}]({url}) — {X} подписчиков

- **Книги в библиотеке:** {books}
- **Упоминания в наших подборках:** {count}
- **Совместные мероприятия:** {если есть}

## Schema.org (для вставки на сайт)

```json
{
  "@context": "https://schema.org",
  "@type": "Person",
  "name": "{Имя Фамилия}",
  "jobTitle": "{Должность}",
  "url": "{site}",
  "image": "{avatar}",
  "sameAs": [
    "{social1}",
    "{social2}"
  ],
  "knowsAbout": [
    "{навык 1}",
    "{навык 2}"
  ],
  "alumniOf": [
    {"@type": "EducationalOrganization", "name": "{ВУЗ}"}
  ]
}
```

## Источники

- [{url1}]({url1}) — {описание}, {дата}
- [{url2}]({url2}) — {описание}, {дата}
```

## Использование

1. Создать файл `experts/{expert-slug}.md`
2. Заполнить плейсхолдеры
3. Добавить в `experts/index.json` (каталог)
4. Опубликовать на сайте (если есть раздел «Эксперты»)
5. Использовать Schema.org JSON-LD для SEO
