# Стоп-слова для поиска экспертов

Используются в `expert-mentions.md` для фильтрации шума и ложных срабатываний.

## Имя автора ≠ эксперт

Когда ищем эксперта, имя автора книги (которого ищем) — НЕ считается экспертом-рекомендателем.

**Исключения:**
- Другие авторы книг, которые ссылаются на эту книгу
- Учёные, цитирующие автора

## Чёрный список для экспертов

Слова-маркеры, которые НЕ считаются экспертной активностью:
- "впечатления", "отзыв", "рецензия", "мнение"
- "прочитал", "прочитала", "купил", "купила"
- "обложка", "бумага", "переплёт", "шрифт"
- "доставка", "упаковка", "продавец"
- "скидка", "акция", "промокод"
- "копия", "пиратка", "скачать бесплатно"

## Стоп-слова для нишевого поиска

### Психология (YMYL!)
- "гадалка", "таролог", "экстрасенс", "ясновидящая"
- "приворот", "порча", "сглаз"
- "магия", "заговор", "ритуал"
- "астролог", "нумеролог"
- "целитель", "энергопрактик"

⚠️ **YMYL-предупреждение:** если эксперт упоминается вместе с этими словами, нужен особый дисклеймер.

### Здоровье (YMYL!)
- "народная медицина", "лечит", "исцеляет"
- "без лекарств", "без врачей"
- "панацея", "100% результат"

⚠️ Для здоровья требуется **обязательная** проверка лицензии / медицинского образования.

### Бизнес / финансы
- "гарантированный доход", "пассивный доход без вложений"
- "финансовая пирамида", "инвестиции с доходом 100%"
- "лёгкие деньги", "заработок в интернете без опыта"
- "форекс-кухня", "бинарные опционы"

## Стоп-слова для языков

- Транслит: `vadim zeland`, `transerfing` — только в контексте автора
- Однофамильцы: `Зеланд` (Нидерланды) ≠ Вадим Зеланд

## Фильтр по эпохе

Эксперт должен быть активен **в то же время**, что и книга (если применимо):
- 1990-2000 → "классика", "переиздание"
- 2000-2010 → "современные подходы"
- 2010-2020 → "актуальные исследования"
- 2020+ → "свежий взгляд", "тренды"

## Бесполезные источники

Для экспертов НЕ используем:
- `aliexpress.com`, `wildberries.ru`, `ozon.ru/product` — магазины
- `pinterest.com`, `instagram.com` — визуальный контент
- `facebook.com` (закрыт в РФ)
- `twitter.com` (закрыт в РФ, низкое качество)
- `livejournal.com` (заброшен)

## Эвристики

### Плохой сигнал (множитель 0.3)
- 1 упоминание, без контекста
- Короткая цитата без ссылки на работу
- Спам-площадка (otzovik, pikabu)
- Имя автора без регалий

### Средний сигнал (множитель 0.7)
- 2-3 упоминания в разных местах
- Есть ссылка на книгу/статью
- Средний по охвату ресурс

### Хороший сигнал (множитель 1.0)
- 4+ упоминания в качественных источниках
- Эксперт цитирует книгу в своей работе
- Совместные мероприятия / интервью
- Переводчик / редактор / рецензент книги

## Проверочные вопросы

Перед добавлением эксперта в карточку:
- [ ] Указаны регалии / место работы / публикации
- [ ] Минимум 2 независимых упоминания
- [ ] Нет маркеров из YMYL-чёрного списка
- [ ] Источники из приоритетного списка
- [ ] Контекст позитивный или нейтральный (не критика)
