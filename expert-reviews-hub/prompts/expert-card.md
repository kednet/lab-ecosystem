# Карточка эксперта (Person/Schema)

## Назначение
Сбор полной карточки эксперта: биография, регалии, контакты, ресурсы, связь с WL.

## Вход
- **Имя эксперта** (обязательно)
- **Контекст** (опционально): тема, ниша, страна, роль (автор/коуч/психолог)

## Алгоритм

### Шаг 1. Определи роль эксперта
- 📚 **Автор** — пишет книги
- 🧠 **Психолог/психотерапевт** — практикует
- 🎓 **Коуч/тренер** — ведёт курсы/тренинги
- 🎙 **Блогер/лидер мнений** — контент в соцсетях
- 🔬 **Учёный/исследователь** — научные работы

От роли зависит, какие регалии важны.

### Шаг 2. Собери базовые данные

#### 2.1. WebSearch — найди официальный сайт
- Запросы:
  - `"{Имя}" официальный сайт`
  - `"{Имя}" биография`
  - `"{Имя}" интервью`
  - `"{Имя}" "{специализация}"`
- Выбери 3-5 авторитетных источников

#### 2.2. WebFetch — спарси каждый источник
Извлеки:
- ФИО полностью
- Фото (URL)
- Дата рождения
- Образование
- Карьера (хронология)
- Специализации
- Регалии (дипломы, сертификаты, звания)
- Публикации (книги, статьи)
- Контакты (сайт, VK, TG, email)
- Цитаты (если есть)

#### 2.3. Для РФ-экспертов дополнительно
- Проверь на b17.ru, psysovet.ru, 7ya.ru
- Проверь в Едином реестре психологов (если психолог)
- Проверь ИНН/ОГРН (если есть ИП/ООО)
- Проверь СНИЛС (если госслужащий — но это публично)

### Шаг 3. Связь с WL (если применимо)

### Шаг 4. Сгенерируй карточку

#### Структура JSON
```yaml

short_bio: "2-3 предложения для карточки"

  - title: "Трансерфинг реальности"
```

См. `templates/expert-schema.json`:
```json

  "description": "...",
  "image": "...",
  "jobTitle": "...",
  "worksFor": {"@type": "Organization", "name": "..."},
  "url": "...",

    "https://vk.com/...",

  "alumniOf": [{"@type": "EducationalOrganization", "name": "..."}]
```

### Шаг 6. Отчёт
`experts/{slug}.report.md`:
- Список всех источников
- Найденные данные + пробелы
- Что нужно добавить
- Рекомендации по использованию

## Выходной формат ответа

```markdown
## Карточка эксперта: {Имя}

**Slug:** {slug}
**Найдено источников:** {N}
**Связь с WL:** {N книг, N цитат}

### ✅ Готово
- Базовая биография
- Фото, контакты
- {N} регалий
- Schema Person

### ⚠️ Требует уточнения
- Год рождения (не найден)
- Email (только через форму)
- ...

### 📁 Файлы
- `experts/{slug}.json`
- `experts/{slug}.md`
- `experts/{slug}.schema.json`
- `experts/{slug}.report.md`

### 🔗 Следующие шаги
1. `/expert mentions {Имя}` — найти упоминания в СМИ
2. `/expert resources {Имя}` — собрать каталог книг/курсов
3. `/expert link {Имя}` — связать с WL
```

## Этические ограничения
- **Не собираем личные данные** (домашний адрес, личный телефон, паспорт)
- **Email — только публичный** с сайта эксперта
- **Фото — только с указанием источника** и лицензии
- **Цитаты — с указанием источника** и не вырываем из контекста
- **Не делаем медицинских/психологических выводов** об эксперте (только факты)
