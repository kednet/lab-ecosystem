# Промпт: `/experts edit {slug}` — ручная правка черновика

## Назначение

Помочь пользователю дозаполнить черновик карточки эксперта перед деплоем. Показываем что есть, подсвечиваем пустые обязательные поля, принимаем точечные правки.

## Вход

- `{slug}` — slug эксперта (например, `mark-rozin`)
- Опциональные правки от пользователя: «добавь секцию X», «замени Y на Z», «готово»

## Алгоритм

### Шаг 1. Прочитай файл

```
experts/{slug}.md
```

Если нет → ошибка «сначала `/experts add`».

### Шаг 2. Проверь status

- `status: draft` → можно править
- `status: published` → предупреди «уже опубликован, всё равно правишь? (правка потребует `/experts deploy`)»
- Нет `status` → добавь `status: draft` в frontmatter

### Шаг 3. Валидация через парсер

```python
from loaders.experts import load_expert
card = load_expert('{slug}')
```

### Шаг 4. Покажи что есть и что пусто

**Критичные поля (без них страница `/experts/{slug}/` будет пустой или с поломанным рендером):**

| Поле | Где в .md | Эффект пустоты |
|------|-----------|----------------|
| `name` | frontmatter `name:` | Заголовок = slug → заглавные буквы через `.title()` (баг!) |
| `tags` (хотя бы 1) | frontmatter `tags:` | На превью `expert-card__tags` пусто |
| `image` | Schema.org `image` или `![...](...)` | Fallback-аватар с первой буквой имени |
| `description` | под `# Имя` или `> цитата-эпиграф` | Hero без описания |
| `jobTitle` (хотя бы 1 слово) | первый `**...**` после `# Имя` или столбец `**Должность / главная роль**` | Подзаголовок = пусто |
| Хотя бы 1 цитата | секция `## 💬 Цитаты` | Блок «Цитаты» не отрисуется |
| Хотя бы 1 соцсеть | Schema.org `sameAs` или секция `## 🎙️ Медиа` | Hero без кнопок ВК/TG/YT |

**Nice-to-have (можно оставить пустыми):**
- `score` (по умолчанию 0, не отображается если 0)
- `knowsAbout`, `alumniOf`, `awards` (рендерится только если не пусто)
- `worksFor`, `email` (рендерится только если не пусто)
- `recommended_books` (если есть — отрисуется, если пусто — секция «Рекомендует книги» не покажется)

### Шаг 5. UX правок

Когда пользователь говорит что менять:

| Фраза пользователя | Что делать |
|--------------------|-----------|
| «добавь тег X» / «удали тег X» | Edit frontmatter `tags:` |
| «опиши как X» / «замени bio на X» | Edit абзац `> ...` после `# Имя` (или первый абзац body) |
| «должность — X» | Edit первый `**...**` после `# Имя` ИЛИ столбец `Должность / главная роль` в `## 📋 Основное` |
| «добавь цитату: ...» | Дописать блок `> «...»\n> — ..., год` в `## 💬 Цитаты` |
| «добавь соцсеть: ...» | Дописать `- [VK](url)` в `## 🎙️ Медиа` ИЛИ в Schema.org `sameAs` |
| «готово» | Edit frontmatter `status: draft` → `status: published` |
| «отмена» / «удали» | `experts/{slug}.md` в trash, выход |

### Шаг 6. После правки — повторная валидация

Прогони `load_expert('{slug}')` ещё раз. Покажи diff:

```
✅ Поле name: "Марк Розин"
✅ Поле description: добавлено
✅ Поле jobTitle: "Партнёр ЭКОПСИ Консалтинг"
⚠️ Поле image: всё ещё пусто (будет fallback-аватар)
⚠️ Тегов: 0 (на превью пусто)
✅ Цитата: 1 шт
✅ Соцсетей: 3 ссылки

👉 Можно деплоить: /experts deploy {slug}
```

## ⚠️ Что НЕ делаем

- ❌ Автоматически НЕ ищем регалии/книги/фото — пользователь даёт ссылки или текст
- ❌ Не переписываем био через LLM целиком — точечно
- ❌ Не деплоим из `/experts edit` — отдельная команда
