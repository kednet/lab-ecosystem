# Связь эксперта с книгами в WishLibrarian

## Назначение
Найти все упоминания эксперта в обработанных книгах WL и создать перекрёстные ссылки.

## Вход
- **Имя эксперта**
- (опционально) путь к WL output, по умолчанию `C:\Users\kfigh\wish_librarian\output\`

## Алгоритм

### Шаг 1. Grep по обработанным книгам

```bash
# Найти все упоминания в конспектах
grep -ril "{Имя}" C:\Users\kfigh\wish_librarian\output\*\summary.md
grep -ril "{Имя}" C:\Users\kfigh\wish_librarian\output\*\workbook.md
grep -ril "{Имя}" C:\Users\kfigh\wish_librarian\output\*\practical_tips.md
```

### Шаг 2. Для каждого совпадения

#### 2.1. Извлечь контекст
- Абзац ± 200 символов вокруг упоминания
- Сохранить дословно (с оформлением)

#### 2.2. Определить роль

| Роль | Маркеры |
|------|---------|
| **author** | "автор книги", "написал", ФИО в заголовке |
| **cited** | "Зеланд пишет:", "по мнению Зеланда", "Зеланд утверждает" |
| **referenced** | "как отмечает Зеланд", "Зеланд в своей книге" |
| **mentioned** | просто упомянуто без цитаты |
| **criticized** | "в отличие от Зеланда", "Зеланд ошибается" |
| **recommended** | "рекомендую Зеланда", "начните с Зеланда" |

#### 2.3. Сохранить цитату
Если есть дословная цитата в кавычках рядом с упоминанием — выделить.

### Шаг 3. Сгруппировать по книге

```yaml
# wl-links/{slug}.json
expert: "Вадим Зеланд"
total_books: 3
total_mentions: 47
total_citations: 12

books:
  - title: "Трансерфинг реальности"
    author: "Вадим Зеланд"
    slug: "transerfing-realnosti"
    role: "author"  # основной автор
    mentions_count: 35
    citations_count: 12
    contexts:
      - role: "author"
        text: "Вадим Зеланд — автор метода управления реальностью через мысли"
        file: "summary.md"
        position: "intro"
      - role: "author"
        text: "Зеланд вводит понятие «пространства вариантов»..."
        file: "summary.md"
        position: "section: Ключевые идеи"
    topics: ["визуализация", "маятник", "пространство вариантов"]
  
  - title: "Сила настоящего момента"
    author: "Экхарт Толле"
    slug: "sil-nastoyashchego"
    role: "cited"
    mentions_count: 8
    citations_count: 0
    contexts:
      - role: "cited"
        text: "В отличие от Трансерфинга Зеланда, Толле предлагает..."
        file: "summary.md"
    topics: ["осознанность", "настоящий момент"]
  
  - title: "[Ещё книга, где упомянут]"
    ...

top_topics:
  - topic: "визуализация"
    count: 15
  - topic: "осознанность"
    count: 12
```

### Шаг 4. Markdown-отчёт

`wl-links/{slug}.md`:

```markdown
# Связи с WL: {Имя}

**Книг в WL:** {N} | **Упоминаний:** {N} | **Цитат:** {N}

## 📚 По книгам

### Трансерфинг реальности (Вадим Зеланд)
**Роль:** автор
**Упоминаний:** 35 | **Цитат:** 12

#### Контексты

**Раздел: Введение**
> Вадим Зеланд — автор метода управления реальностью через мысли.

**Раздел: Ключевые идеи**
> Зеланд вводит понятие «пространства вариантов»...

#### Темы в этой книге
- визуализация (15)
- маятник (12)
- пространство вариантов (8)
- намерение (7)

---

### Сила настоящего момента (Экхарт Толле)
**Роль:** cited
**Упоминаний:** 8

#### Контексты
> В отличие от Трансерфинга Зеланда, Толле предлагает жить в настоящем...

#### Темы
- осознанность (5)
- настоящий момент (3)

## 🔝 Топ тем

1. визуализация — 15
2. осознанность — 12
3. маятник — 12
4. пространство вариантов — 8

## 💡 Рекомендации

- **Для карточки эксперта на странице книги** — используй 3-5 самых ярких цитат
- **Для лендинга** — собери 10 цитат из разных книг
- **Для email-курса** — темы по убыванию частотности
- **Для WishCoach** — если эксперт — коуч, добавь в библиотеку тренера
```

### Шаг 5. Обновление метаданных WL

Для каждой связанной книги добавь в `metadata.json`:
```json
{
  "experts_mentioned": [
    {
      "slug": "vadim-zeland",
      "name": "Вадим Зеланд",
      "role": "author",
      "mentions": 35,
      "citations": 12
    }
  ]
}
```

Это позволит потом:
- На странице книги показать блок «Другие книги этого автора в WL»
- В карточке эксперта — список книг в WL
- Связать `/reviews` с автором через `author_slug`

## Использование
- Карточка эксперта на лендинге WishCoach
- Блок «Об авторе» на странице книги
- Кросс-ссылки между книгами одного автора
- Рекомендательная система (если понравилась книга X, посмотри Y того же автора)

## Пример вызова
```
/expert link "Вадим Зеланд"
/expert link "Борис Ананьев" --path "C:\Users\kfigh\wish_librarian\output\"
/expert link "Марк Розин"
```
