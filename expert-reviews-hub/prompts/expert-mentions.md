# Сбор упоминаний эксперта в интернете

## Назначение
Найти ВСЕ упоминания эксперта в интернете: пресса, подкасты, YouTube, форумы, цитирование в книгах.

## Вход
- **Имя эксперта**
- (опционально) тема/ниша для фильтрации
- (опционально) диапазон дат

## Алгоритм

### Шаг 1. WebSearch — серия запросов

#### 1.1. Пресса
```
"{Имя}" интервью
"{Имя}" статья
"{Имя}" "{Издание}"  # Коммерсант, РБК, Forbes, Ведомости
"{Имя}" журнал
```

#### 1.2. Подкасты
```
"{Имя}" подкаст
"{Имя}" podcast
"{Имя}" выпуск
"{Имя}" аудио
```

#### 1.3. YouTube
```
"{Имя}" youtube
"{Имя}" видео
"{Имя}" интервью видео
"{Имя}" лекция
```

#### 1.4. Соцсети и форумы
```
"{Имя}" вконтакте
"{Имя}" telegram
"{Имя}" pikabu
"{Имя}" vc.ru
"{Имя}" habr
```

#### 1.5. Цитирование (в книгах)
```
"{Имя}" цитаты
"как говорил {Имя}"
"по словам {Имя}"
"по мнению {Имя}"
```

### Шаг 2. Фильтрация

Убирай:
- ❌ **SEO-спам** (дорвеи с 10+ вхождениями имени)
- ❌ **Случайные совпадения** (другие люди с тем же ФИО)
- ❌ **Каталоги без контекста** (просто список без упоминания)
- ❌ **Дубликаты** (тот же материал на разных площадках — оставляй 1)
- ❌ **Старые > 5 лет** (если не asked for historical)

Оставляй:
- ✅ **Реальные интервью/статьи** (с цитатой или пересказом)
- ✅ **Обзоры книг/курсов** эксперта
- ✅ **Упоминания в новостях** (контекст события)
- ✅ **Видео с экспертом** (интервью, лекции)
- ✅ **Цитирование в чужих статьях** (с автором)
- ✅ **Упоминания в книгах** (с ISBN/названием)

### Шаг 3. Скоринг

| Критерий | Вес |
|----------|-----|
| Домен (DA/DR) | 0-30 |
| Объём упоминания | 0-20 |
| Прямая цитата | 0-20 |
| Дата (свежесть) | 0-15 |
| Уникальность | 0-15 |
| **Итого** | **0-100** |

#### Домены с высоким DA (примеры для РФ)
- ria.ru, tass.ru, kommersant.ru, rbc.ru, vedomosti.ru
- forbes.ru, russia24.ru
- vc.ru, habr.com
- meduza.io, dw.com (иностранные о РФ)

#### Домены со средним DA
- pikabu.ru, woman.ru
- Лайфхакер, AdMe, BroDude

### Шаг 4. Категоризация

| Категория | Иконка | Маркеры |
|-----------|--------|---------|
| Пресса | 📰 | домен СМИ, формат статьи |
| Подкаст | 🎙 | podcast, audio, episode |
| YouTube | 📺 | youtube.com, youtu.be |
| Telegram | 📱 | t.me/... |
| VK | 💬 | vk.com/... |
| Форум | 💭 | pikabu, vc, habr, forum |
| Книга | 📚 | isbn, publisher, страница книги |
| Научная | 🔬 | doi, researchgate, cyberleninka |

### Шаг 5. Bundle

```yaml
# mentions/{slug}.json
expert: "Вадим Зеланд"
total_mentions: 47
categories:
  press: 12
  podcast: 3
  youtube: 18
  telegram: 5
  vk: 4
  forum: 2
  book: 3
date_range:
  earliest: "2015-03-12"
  latest: "2026-05-28"
mentions:
  - url: "https://ria.ru/..."
    title: "..."
    source_domain: "ria.ru"
    category: "press"
    date: "2026-05-12"
    score: 87
    snippet: "Прямая цитата или абзац"
    type: "interview"  # interview | review | citation | news
```

## Эвристики
- **Сначала ищи цитаты** — они самые ценные
- **Потом интервью** — раскрывают эксперта
- **Потом обзоры** — мнение других
- **Потом упоминания в новостях** — контекст
- **Форумы** — живые обсуждения, цитаты читателей
