# AI-суммаризация отзывов (pro/cons, рейтинг, тренды)

## Назначение
Превратить 100+ сырых отзывов в **структурированную аналитику**: pros/cons, средний рейтинг, тренды, цитаты, verdict.

## Вход
- **Bundle** отзывов (JSON из любого источника)
- (опциональчно) фокус: только pro / только con / нейтрально

## Алгоритм

### Шаг 1. Подготовка данных

```python
# Из bundle.json собрать все тексты + мета
reviews = bundle["reviews"]  # list of {author, rating, text, source, ...}
sources = bundle["sources"]  # weights для каждого источника
```

### Шаг 2. Скоринг источников

Применить вес (из `data/sources-rating.md`):

```yaml
# data/sources-rating.md
sources:
  litres: 1.5    # verified покупатели
  own: 1.5       # наши клиенты
  livelib: 1.2   # крупнейшая база
  ozon: 0.7      # спам снижает
  author_today: 1.0
  goodreads: 1.0
  vk: 0.8        # живой, но без рейтинга
  telegram: 0.7
  youtube: 0.9   # развёрнуто
```

### Шаг 3. AI-анализ текстов

Используем AI для:

#### 3.1. Классификация тональности
Для каждого отзыва:
- **positive** — хвалит
- **neutral** — нейтрально описывает
- **negative** — ругает
- **mixed** — и плюсы, и минусы

#### 3.2. Извлечение pros/cons

Группируем по темам:

**Pros (3-7 пунктов) по убыванию частоты:**
1. "Изменяет мышление / отношение к реальности" — 234 упоминания (78%)
2. "Практические техники, применимые в жизни" — 198 (66%)
3. "Простой язык, легко читается" — 167 (55%)
4. "Мотивирует к действию" — 145 (48%)
5. "Подходит для начинающих" — 89 (30%)

**Cons (3-7 пунктов):**
1. "Нет научного обоснования" — 156 (52%)
2. "Повторы в серии книг" — 89 (30%)
3. "Местами вода, растянуто" — 67 (22%)
4. "Слишком эзотерично" — 45 (15%)

#### 3.3. Топ-цитаты
- Самые яркие высказывания (1-3 предложения)
- С указанием автора и источника
- 3-5 штук

#### 3.4. Тренды
Динамика по времени:
- 2024: 8 отзывов/мес
- 2025: 12 отзывов/мес (+50%)
- 2026: 15 отзывов/мес (+25%)

Тренд: **растёт**

#### 3.5. Скоринг доверия
- Количество отзывов
- Доля verified
- Распределение оценок (если 90% пятёрок — подозрительно)
- Разброс в текстах (если все одинаковые — спам)

### Шаг 4. Финальный verdict

1-2 предложения для карточки книги:
> «Средняя оценка 4.7/5 на основе 1247 отзывов. Книга хвалят за практичность и изменение мышления, но критикуют за отсутствие научной базы. Подходит для тех, кто готов к нестандартному подходу.»

## Использование

1. **Карточка книги** — verdict + рейтинг + 3-5 цитат

3. **Email-рассылка** — топ-3 pros для подписчиков
4. **Лендинг курса** — цитата + рейтинг для социального доказательства
