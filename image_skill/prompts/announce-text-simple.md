# Announce Text — SIMPLE (retry, v1.3)

Упрощённая версия. Если основной промпт дал сбой — пробуем этот.

## Задача
Напиши 4 адаптации поста на русском по заголовку «{title}» для сообщества «{display_name}».

## Каналы
- **vk**: 3 строки + эмодзи, до 600 символов, hashtags в конце
- **tg**: 1 абзац + ссылка, HTML `<b>`, до 800 символов
- **ok**: 2 абзаца, без эмодзи, до 600 символов
- **zen**: 4-6 абзацев, без hashtag, 500-1500 символов

## Данные
- Хэштеги: {hashtags}
- Ссылка: {live_url}

## Формат ответа

JSON без markdown, начинай с `{{`:

```json
{{"vk": "...", "tg": "...", "ok": "...", "zen": "..."}}
```
