## Структура

### Заголовок (вверху страницы)
- H1 "Карта желаний" 48pt DM Serif Display `#881337` по центру
- Опционально скрипт "моя" 28pt Dancing Script `#E11D48` справа
- Y-отступ от верха рабочей области: **15 мм**

### 6 секторов (2 ряда × 3 колонки)

| Параметр | Значение |
|---|---|
| Размер сектора | **~130 × 120 мм** |
| Зазор между секторами | **5 мм** |
| Скругление углов | **8 мм** |
| Толщина рамки | **0.5 мм** цвет сектора opacity 0.6 |
| Фон сектора | **сплошной цвет** (для печати) или **градиент** (для экрана) |

### Цвета секторов
Берём из `_style-guide.md` §2:
- 💪 Здоровье: `#10B981` (или градиент `#34D399` → `#059669`)
- ❤️ Семья: `#EC4899` (или `#F472B6` → `#BE185D`)
- 💰 Финансы: `#F59E0B` (или `#FBBF24` → `#B45309`)
- 🚀 Карьера: `#3B82F6` (или `#60A5FA` → `#1D4ED8`)
- 🧘 Осознанность: `#8B5CF6` (или `#A78BFA` → `#6D28D9`)
- 🏖️ Отдых: `#06B6D4` (или `#22D3EE` → `#0E7490`)

### Содержимое сектора (пустое)

```
┌──────────────────────────┐
│                          │
│  💪  Здоровье            │  ← emoji 32pt + название 18pt Manrope 700 белый
│                          │
│  ┌────────────────┐      │
│  │                │      │  ← пустой блок для фото
│  │                │      │     60×40 мм
│  │                │      │     рамка 0.3мм белый
│  └────────────────┘      │
│                          │
│  Желание 1: __________   │  ← 3 строки для текста
│  Желание 2: __________   │
│  Желание 3: __________   │
│                          │
│  Срок: ________          │  ← поле срок (опц.)
│                          │
└──────────────────────────┘
```

**Детали:**
- Emoji + название: Y-отступ от верха **12 мм**, белый текст
- Блок для фото: 60×40 мм, отступ сверху **15 мм** от названия, по центру сектора, рамка `0.3 мм` цвет `#FFFFFF` opacity 0.7
- "Желание N:" — Y-отступ от блока фото **8 мм**, 12pt Manrope 700 белый
- Линии для текста: 0.2 мм `#FFFFFF` opacity 0.5, длина **80 мм**, шаг **8 мм**
- "Срок:" — 1 строка внизу, мелко 10pt белый opacity 0.7
